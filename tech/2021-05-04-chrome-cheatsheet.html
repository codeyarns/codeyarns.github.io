Though my primary browser is Firefox, I do keep Google Chrome around as a backup for webpages that do not work correctly with the former. Here are some tips and tricks that I find useful with Chrome.

  • Disable notifications: Open this link to toggle notifications and control it at a per-domain basis. Note that the link only works with Chrome.

  • Search bookmarks: I like to search bookmarks using *, just like in Firefox. To do this, go to Settings -> Search engine and add a new search engine. Use * as the keyword and provide the search URL of the form chrome://bookmarks/?q=%s

  • Search history: I like to search history using ^, just like in Firefox. To do this, go to Settings -> Search engine and add a new search engine. Use * as the keyword and provide the search URL of the form chrome://history/?q=%s

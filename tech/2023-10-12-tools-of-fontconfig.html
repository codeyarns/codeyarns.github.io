Tools of FontConfig

The FontConfig library ships with several tools that are essential to learning and managing the fonts on Linux.

List

  • To list all the installed fonts on the system:
$ fc-list
  • To list a particular font:
$ fc-list "Ubuntu Mono"

An example line from the output shows the path to the font file, the font name and its style:

/usr/share/fonts/truetype/ubuntu/UbuntuMono-R.ttf: Ubuntu Mono:style=Regular

Dump

  • To dump all the information about all the installed files from their cache files:
$ fc-cat

Match

  • To find out which font would be chosen by fontconfig by applying its matching rules for a given input string:
$ fc-match "Ubuntu Mono"
UbuntuMono-R.ttf: "Ubuntu Mono" "Regular"

Cache

  • To add fonts to the fontconfig cache, copy the font files to ~/.fonts and run this command:
$ fc-cache -fv

This installs the font files to $HOME/.local/share/fonts.

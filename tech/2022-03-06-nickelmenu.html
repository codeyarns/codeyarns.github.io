NickelMenu is open source software for Kobo ebook readers, like my Kobo Nia, that can be used to add custom menu items to all the menus in Koboâ€™s eReader software.

Installation

  • Download the latest KoboRoot.tgz from the releases page.
  • Connect the Kobo ebook reader using USB cable to your computer and click Connect on the Kobo. It should appear as a KOBOeReader drive in Windows Explorer.
  • Copy the downloaded file to the KOBOeReader\.kobo directory.
  • Eject the Kobo safely and it should reboot and install the NickelMenu software.

You should be able to see the default NickelMenu menu in various menu options in the Kobo now.

Configuration

  • Connect the Kobo again like above to the computer and open/create the KOBOeReader\.adds\nm\config file.
  • Consult the documentation on how to write your own config file.
  • Edit the config file to have the menu items you want in the NickelMenu that appears in the main screen main, the ebook reader reader, the library library and the browser browser.
  • Eject the Kobo and the effect of your config should be immediately visible in the NickelMenu menus in the reader.

I keep my config restricted to just the extra menu items I want:

menu_item :main    :Pocket              :nickel_open: library: pocket
menu_item :main    :Invert Screen       :nickel_setting :toggle :invert
menu_item :main    :> Portrait          :nickel_orientation :portrait
menu_item :main    :> Landscape         :nickel_orientation :landscape
menu_item :main    :> Reverse Portrait  :nickel_orientation :inverted_portrait
menu_item :main    :> Reverse Landscape :nickel_orientation :inverted_landscape

How to view or hide comments in LibreOffice Writer

Ashwin Nanjappa

2013-07-17 03:46:56+00:00

LibreOffice Writer can be used to collaborate with another person or a team to create a document. You can view the comments written by others for you in the document by choosing View > Comments.

If there are too many comments or you find them distracting while you edit your document, you can hide them using the same method: View > Comments.

Tried with: LibreOffice 3.5.7.2 and Ubuntu 12.04 LTS

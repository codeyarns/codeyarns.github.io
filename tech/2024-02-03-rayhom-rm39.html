RayHom RM-39 Battery Charger

📅 2024-Feb-03 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ battery ⬩ 📚 Archive

RayHom RM-39 is a charger for rechargeable NiMH batteries. It handles all the battery formats that are typically used at home: AAA, AA, C, D and 9V.

Battery types

The number of batteries of each type that can be charged at the same time:

  • AAA: 4 x 1.2V
  • AA: 4 x 1.2V
  • C: 4 x 1.2V
  • D: 4 x 1.2V
  • 9V Cell: 2 x 9V

Notes

  • There is a nice LCD display on the front that displays the current charge level of each of the batteries. It flashes while the battery is being charged and shows a non-flashing steady display when the battery is done being charged.

  • Its charging speed of the common AAA and AA batteries is pretty slow. My Panasonic BQ-CC55 can charge these types in a fraction of the time.

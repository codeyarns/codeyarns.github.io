How to stop websites from messing with Firefox scrollbars

📅 2019-Sep-14 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ firefox, scrollbar ⬩ 📚 Archive

Problem

Of late, websites have begun to style the scrollbars displayed inside Firefox. They try to change the width and color of the scrollbars. This usually makes the scrollbar hard to grab and drag with the mouse.

Solution

By default, Firefox uses the scrollbar style of your OS or windowing environment for its websites. Websites manipulate the scrollbar using the CSS properties scrollbar-color and scrollbar-width.

To take back control, open about:config and set the following two settings to false:

layout.css.scrollbar-color.enabled
layout.css.scrollbar-width.enabled

Tried with: Firefox 69.0 and Windows 10

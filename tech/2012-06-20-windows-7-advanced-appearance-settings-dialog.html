After Aero was introduced in Vista and Windows 7, the configuration of how a window is drawn has been hidden away. This includes configuration of the size, color and display of all parts of a window: titlebar, border, icons, scrollbar, tooltip and so on.

The old dialog which used to handle this is still there, it is now called Window Color and Appearance. To access it, right-click on Desktop, choose Personalize ‚Üí Window color ‚Üí Advanced appearance settings.

The command to invoke this dialog directly is "C:\Windows\System32\rundll32.exe" shell32.dll,Control_RunDLL desk.cpl,Advanced,@Advanced

Tried with: Windows 7 64-bit

LaTeX Error: lineno.sty not found

2014-05-14 08:25:46+00:00

Problem

LaTeX Error: File `lineno.sty' not found.

Solution

The lineno.sty is shipped in the texlive-humanities package. Install it to fix this error:

$ sudo apt-get install texlive-humanities

Tried with: Ubuntu 14.04

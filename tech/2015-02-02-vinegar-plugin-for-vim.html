Vinegar plugin for Vim

2015-02-02 12:03:34+00:00

[caption id="attachment_5644" align="aligncenter" width="576"][](https://codeyarns.files.wordpress.com/2015/02/20150202_vinegar.png) Vinegar plugin for Vim[/caption]

Vinegar is a plugin written by the famous Tim Pope for Vim. It enhances the built-in NetRW plugin to make it easier to navigate the filesystem and open files. It can be installed from its Github page here.

Press - in any open file. NetRW buffer opens up showing the files in the directory of the open file. Keep pressing - to move up the directory hierarchy. Press j and k to move up and down the files and Enter to open directories and files.

Tried with: Vim 7.4 and Ubuntu 14.04

Starship Cross-Shell Prompt

📅 2022-Dec-03 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ shell ⬩ 📚 Archive

Starship is a program that provides a colorful customizable prompt for your favorite shell. Development happens as open source over at https://github.com/starship/starship. Both the shells that I use, fish and bash, are supported by this tool!

Install

  • Download the Starship binary and place it in /usr/local/bin:
$ curl -sS https://starship.rs/install.sh | sh
  • Add the invocation at the end of fish and bash shell config files:
# ~/.config/fish/config.fish
starship init fish | source
# ~/.bashrc
eval "$(starship init bash)"
  • Restart your shell and bingo there’s a colorful fancy prompt! ✨

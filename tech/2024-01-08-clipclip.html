ClipClip clipboard manager

📅 2024-Jan-08 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ clipboard ⬩ 📚 Archive

ClipClip is a clipboard manager for Windows.

  • Install: Download and install it from here.

  • Usage: It starts with Windows by default and runs in the background in the notification area capturing all the items that are copied.

  • Configuration: This tool can be configured by opening its manager and going to Options -> Configuration.

  • Keyboard shortcut: The default shortcut to paste from this tool is Ctrl + Shift + v. Ideally, I wanted to change it to the Windows clipboard Win + v, but it would not accept this shortcut. The default shortcut is a bit of a problem in apps which actually use it, like Terminal for example.

Tried with: ClipClip Pro 2.4.5874

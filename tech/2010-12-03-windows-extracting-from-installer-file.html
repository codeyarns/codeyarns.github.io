Why?

Sometimes you want to pull out a few files from inside the installer or setup file of a program. That is, to extract a few files from the .exe or .msi installer files of a program.

How?

  • 7-Zip is not only the best compression tool on Windows, it can also be used to peek into and extract from most installer files! :-) A large number of installer files are self-extracting archives or Windows installers or installers created using open-source tools like NSIS. 7-Zip supports uncompressing most of these formats.

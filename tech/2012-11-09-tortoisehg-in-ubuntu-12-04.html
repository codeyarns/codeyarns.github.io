TortoiseHg in Ubuntu 12.04

📅 2012-Nov-09 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ thg, tortoisehg, ubuntu ⬩ 📚 Archive

TortoiseHg is available on Ubuntu 12.04. However, integration with the file explorer, in this case Nautilus, that it is famous for does not seem to be working. Thankfully, it can still be launched from the terminal like in Windows.

Tried with: TortoiseHg 2.2

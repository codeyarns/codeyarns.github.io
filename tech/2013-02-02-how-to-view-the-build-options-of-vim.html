How to view the build options of Vim

2013-02-02 14:22:50+00:00

Many Vim configuration settings and plugins are dependent on certain build options. These are options that were enabled when the Vim executable you are using was built. To view the build options inside Vim use :version

The build options are a mass of text and it is hard to look for a certain option in it. If you are looking for a particular build option, then try vim --version at the commandline, which gives the same output as above. But, this can be piped to grep to search for whatever you want.

Tried with: Vim 7.3 and Ubuntu 12.04 LTS

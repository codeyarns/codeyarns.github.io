csv.vim plugin

Ashwin Nanjappa

2013-12-24 12:13:11+00:00

If your programs produce CSV files and you need to view and edit them, then you will love the csv.vim plugin for Vim. It can be installed from its Github repository here.

After installation, just open any .csv file in Vim and it will display it like a spreadsheet with columns and rows and colors. You can further use commands to make all the columns equal width and so on. It makes viewing CSV files from the terminal a lot of fun! :-)

Tried with: Vim 7.3 and Ubuntu 12.04

NetSurf

(First posted on: 2015-10-21 12:10:00+00:00)

NetSurf is one of the many alternate browsers claiming to deliver speed and simplicity when compared to the big slow browsers.

An old version of NetSurf is available on Ubuntu archives:

$ sudo apt install netsurf

  • It seems to be most user friendly and familiar of the tiny browsers.

  • It uses its own rendering engine and frameworks. So, expect not all websites to work well.

  • It was not fast enough for me to use it as a documentation or local HTML webpage viewer.

Tried with: NetSurf 2.9 and Ubuntu 14.04

SysInternals Desktops

2011-05-27 11:32:52+00:00

I am huge fan of the tools from SysInternals, life without them on Windows would be difficult. So, when I learnt that they had a virtual desktop utility named Desktops, I had to check it out.

Desktops is a minimal, but rock-solid virtual desktop manager. It creates a separate Windows desktop object for each new virtual desktop it creates. The original desktop (Desktop 1) is left untouched and 3 additional desktops are created when needed. The usage of a desktop object gives robustness, but leaves it with some serious drawbacks. The 3 virtual desktops do not use the Aero theme settings I have set and also keyboard shortcuts do not seem to work in them for some reason! Also, some applications like Firefox do not seem to be capable of opening a new instance in a separate virtual desktop. Due to these problems, Desktops is simply not an option for me right now.

Tried with: Desktops 1.0.2 and Windows 7

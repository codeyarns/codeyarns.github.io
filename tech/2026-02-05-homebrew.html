Homebrew is a package manager for MacOS to install, remove or update applications/libraries from the shell.

  • Install:

Download and install the latest .pkg installer from the Github releases page.

  • Install a package:

A package is called bottle and installing it is called pouring.

$ brew install some_app

The binaries installed by Homebrew can be seen at /opt/Homebrew/opt.

  • Add a package repository:

Online package repository is called a tap.

$ brew tap some_repo
  • Update package info:

To update package information about Homebrew itself and the installed packages:

$ brew update
  • Upgrade packages:

To actually upgrade the installed Homebrew and packages:

$ brew upgrade
  • Clean:

To clean older packages and information:

$ brew cleanup

Tried with: Homebrew 5.0.13 and MacOS Tahoe 26.2

How to add Google search to website

📅 2019-Sep-08 ⬩ ✍️ Ashwin Nanjappa ⬩ 📚 Archive

Google provides a service called Custom Search which can be used to add search functionality to search your own website. That is, you can place a search bar in your website that your users can use to search only the content of your website using Google's search engine.

  • Head over to Google Custom Search. Add a new search engine by providing your website domain or base URL of your website.
  • Go to the Look and feel section of your search engine to customize how the search bar looks and how its results are displayed.
  • In the Setup section, click on Get code to get the HTML code you can paste into your webpages. It will look something like this:
<script async src="https://cse.google.com/cse.js?cx=blahblahblahblahblah:blahblah"></script>
<div class="gcse-search"></div>
  • Copy the script part into the head section of your HTML webpage. Copy the div part into whereever you want the search bar in the body section of your HTML webpage. Your webpages should now have a Google Custom Search bar displayed where you placed it.
  • That is it. I have found that placing a search bar on my webpages useful to search my own content. Because Google makes finding things so easy!

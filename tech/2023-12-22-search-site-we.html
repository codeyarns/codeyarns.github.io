Search Site WE extension for Firefox

📅 2023-Dec-22 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ firefox extension ⬩ 📚 Archive

Search Site WE is an extension for Firefox that makes it easy to search inside all the webpages of a subdomain or domain using a search engine. It is essentially a shortcut for opening a search engine webpage and using the site:somewebsite.com along with the search keyword.

  • Install: It can be installed from here.

  • Usage: Click the button of the extension in the toolbar or use the keyboard shortcut Alt + k. This opens a dialog to type the keyword, search options and click Enter to search using the search engine.

  • Settings: Go to the settings of the extension to set the default search engine. I like to switch from Google to DuckDuckGo here.

Tried with: Search Site WE 4.7, Firefox 119.0.1 and Ubuntu 22.04

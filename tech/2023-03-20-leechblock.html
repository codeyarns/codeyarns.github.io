LeechBlock

📅 2023-Mar-20 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ firefox ⬩ 📚 Archive

LeechBlock is an extension for Firefox, Chrome and Edge browsers that can be used to block websites that you think are distracting. The websites can be blocked during certain days or times, or delayed or accessed only by typing a password or random phrase. I have only tried this extension on Firefox, which is my primary browser.

  • The addon is called LeechBlock NG for Firefox and can be installed here.

  • Block sets: In the Options section of this addon, various block sets can be created or deleted. Each block set will get its own tab in the Options with the tab title being the custom name you set for the block set.

  • Delay websites:

    • To delay the opening of certain websites, create a new block set with a suitable name, like Delay these websites.
    • Add the websites (wildcards are supported) to the domain names list.
    • Set the time periods and days when the block needs to be in effect.
    • In the How to block section click the Delaying Page option.
    • Set the amount of delay in seconds.

How to install PyTorch

📅 2020-Mar-04 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ ⬩ 📚 Archive

PyTorch can be installed in many ways. I prefer installing the Python3 version locally using pip3 and with CUDA/GPU support.

To do this:

  • Install updated NVIDIA drivers for your GPU.
  • Install latest version of CUDA.
  • Install pip3 if you do not have it:
$ sudo apt install python3-pip
  • Install PyTorch locally:
$ pip3 install --user torch torchvision
  • Check if Torch is working with CUDA support:
$ python3
>>> import torch
>>> torch.cuda.is_available()
True
>>> torch.rand(5)
tensor([0.9035, 0.2334, 0.3102, 0.2368, 0.5607])

Tried with: PyTorch 1.4.0, Python 3.6, CUDA 10.1 and Ubuntu 18.04

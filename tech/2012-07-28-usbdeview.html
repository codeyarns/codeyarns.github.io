USBDeview is a comprehensive tool from NirSoft to learn more about the USB hubs and devices that are connected to your computer. It can also display a history of all the USB devices that were previously connected to the computer.

For each device, it can display a detailed set of USB properties. To display the Vendor ID and Product ID of USB devices, USBDeview requires the usb.ids files which can be downloaded here. Place the file in the same directory as USBDeview executables and it will use the ID information.

USBDeview has many more features than the similarly named USB View, though it cannot display the topography of USB devices.

Tried with: USBDeView 2.11

MyUninstaller

📅 2011-Jan-25 ⬩ ✍️ Ashwin Nanjappa ⬩ 📚 Archive

MyUninstaller is a great tool from NirSoft. I keep it around because of a few good reasons:

  • It shows the complete list of programs whose install information is available in Windows. Control Panel only shows a subset of this list.

  • It shows all the details for each installed program. This includes details that are not displayed in Control Panel. For example: uninstall command string, change command string, install directory and so on.

  • It can be used to uninstall or change any of the installed programs on Windows.

  • And most importantly, if an uninstall fails and its entry is stuck in the list, that entry can be deleted. You cannot do this in Control Panel.

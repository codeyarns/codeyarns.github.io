VSFTPD FTP Server

📅 2022-Dec-20 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ ftp ⬩ 📚 Archive

Very Secure FTP Daemon (VSFTPD) is a popular FTP server for Linux. It can be used for simple file sharing on a network between a Linux server computer and client computers running any OS.

Usage

  • Install: Installing on Ubuntu is easy:
$ sudo apt install vsftpd

Check if the vsftpd service is running successfully:

$ sudo systemctl status vsftpd
  • Connect from client: To connect from the client use these settings:
    • Protocol: FTP (not SFTP)
    • IP address of the server
    • Port: 21
    • Login and password of any existing user on the server

Configuration

Changes to the vsftpd configuration are done by editing its configuration file /etc/vsftpd.conf and restarting the vsftpd service using this command:

$ sudo systemctl restart vsftpd
  • Allow uploads: In the configuration file, uncomment the write_enable=YES line.

Tried with: Ubuntu 22.04 and VSFTPD 3.0.5

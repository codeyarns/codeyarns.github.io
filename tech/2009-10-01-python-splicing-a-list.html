Python: Splicing a List

(First posted on: 2009-10-01 07:18:03+00:00)

Splicing a list in Python means extracting a subsequence from a list

aList = [5, 6, 7, 8, 9]
aList[2:4] # [7, 8] Yes, end index is one more than the end
aList[:4]  # [5, 6, 7, 8]
aList[2:]  # [7, 8, 9]
aList[:]   # Complete list

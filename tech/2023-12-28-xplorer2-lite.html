Xplorer2 is a fantastic professional file manager for users who are not satisfied with Windows Explorer. Xplorer2 Lite is the free light version of that file manager. I find that the features in the Lite version are sufficient for me to replace my use of Windows Explorer.

  • Install: It can be downloaded and installed from here.

  • Dual pane: This is the primary reason I need an alternative for Windows Explorer. The second pane can be toggled using Ctrl + o or choosing View -> Dual Pane.

  • Font: The font used for the UI can be changed at Tools -> Options -> Window -> Main Font (at the bottom-left).

  • Inactive background: I like to clearly known which window is active right now by using a bright color. This can be set at Tools -> Options -> Window -> Inactive background (at the bottom-center).

Tried with: Xplorer2 Lite 5.4.0.2 and Windows 11 x64

How to flush DNS cache in Ubuntu

📅 2014-Jul-06 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ cache, dns, ubuntu ⬩ 📚 Archive

Install the Name Service Cache Daemon (NSCD):

$ sudo apt install nscd

Restart the daemon:

$ sudo service nscd restart

Tried with: Ubuntu 14.04

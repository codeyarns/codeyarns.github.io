GPU Burn is a tool written by Ville Timonen to check if the GPU is stable in its thermals and computation under load. Such issues get hidden with the complexity of running modern workloads.

  • To download and build the tool:
$ git clone https://github.com/wilicc/gpu-burn.git
$ cd gpu-burn
$ make COMPUTE=80
  • To test using tensor cores for 10 minutes:
$ ./gpu-burn -tc 600

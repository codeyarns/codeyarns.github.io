How to dilate or erode image in Octave

(First posted on: 2014-05-12 07:18:29+00:00)

  • Make sure the image package for Octave is installed. It can be installed easily:
$ sudo apt-get install octave-image
  • First create a structuring element to use for dilation or erosion:
$ se = strel("square", 3);
$ se
1 1 1
1 1 1
1 1 1
  • Next, use the structuring element to perform dilation or erosion:
$ B = imdilate(A, se);
$ D = imerode(C, se);

These functions are from Matlab and so will work on it too.

Tried with: Octave 3.8.1 and Ubuntu 14.04

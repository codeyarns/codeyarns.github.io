How to create a Ubuntu installer USB flash drive on Windows

There are many tools that can be used to create a USB installer flash drive from an Ubuntu ISO file.

First, download the ISO file of the Ubuntu you want to install. I prefer to download using BitTorrent for speed and robustness.

Use one of the tools below to create the installer flash drive. Then plug the USB flash drive in, reboot Windows and choose the flash drive while booting up and install Ubuntu from it.

Using Balena Etcher

  • Download Etcher from here.
  • No installation was needed, since I downloaded the portable version for Windows. Windows showed a warning when executing it, but I allowed it to run and there were no further issues.
  • It asks for the .iso file, the USB flash drive to use and then click to flash. Helpfully, it shows which are the system drives and hard disks in the drive choosing UI, so you can avoid making a mistake.
  • After flashing, it even runs a validation check to make sure it was flashed correctly. Though this check said that the Flash Failed!, the flash drive worked fine on installing.

Using LiLi

Note: This tool is no longer maintained.

  • Download and install the Linux Live USB Creator, also called LiLi. It is installed with the name LiLi USB Creator in the Windows Start menu or Start screen.

  • Plug in a USB flash drive of capacity 1GB or more. It should be mounted by Windows as a drive.

  • Run LiLi. Yes, it has a weird and funky GUI. Choose the USB drive and downloaded Ubuntu ISO file for installation. If the settings you chose are correct, you should be able to click the Lightning button at the bottom. LiLi should make a bootable Ubuntu installer on your USB flash drive.

Eclipse: Visual Studio key bindings

📅 2009-Oct-14 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ eclipse, visual studio ⬩ 📚 Archive

I was dreading having to manually modify Eclipse key bindings to Visual Studio ones, which I am used to. Thankfully, Eclipse ships with support for Visual Studio key bindings.

Go to Windows → Preferences → General → Keys and in the Scheme dropdown choose Microsoft Visual Studio.

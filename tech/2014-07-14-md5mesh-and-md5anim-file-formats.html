The MD5Mesh file format is used to represent a creature with skeleton and mesh. The MD5Anim is a companion file format to represent the animations to articulate the body. These file formats were introduced by ID Software with their Doom 3 video game to store and load skeletal animation of characters. These formats are in ASCII, are human readable and easy to understand.

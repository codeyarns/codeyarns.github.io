Since forever, Notepad++ has been my text editor of choice on Windows to take quick/temporary notes, as a copy-paste buffer or as the default editor for random text formats.

  • Set word wrap: Choose View -> Word wrap.

  • Set font style: Go to Settings -> Style Configurator. Chose Global Styles (Language) and Global override (Style) and pick the font name, size and other details. Choose the options such as Enable global font and others to set this as default. Do the same for all the file formats you prefer.

  • Display line number: Go to Settings -> Preferences -> Margins/Border/Edge. Enable in the Line Number section and make your preferences for dynamic/constant width for the line number display.

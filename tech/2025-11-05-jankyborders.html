Something I miss from older versions of Windows is clear indication of which window is currently in focus. In MacOS Tahoe, the only indication of the window in focus is that its traffic light buttons are in color, while they are greyed out for the rest of the windows. JankyBorders is an open-source app that helps draw a colored border around the currently focused window.

  • Install:
$ brew tap FelixKratz/formulae
$ brew install borders
  • Test the app:

To test the app from the terminal with a thick green border:

$ borders width=10 active_color=0xFF00FF00
  • To run as service:

First create a configuration file ~/.config/borders/bordersrc:

#!/bin/bash

options=(
    width=10
    active_color=0xFF00FF00
)

borders "${options[@]}"

Next run as service in the background:

$ brew services start borders

Tried with: JankyBorders 1.8.4 and MacOS Tahoe 26.0.1

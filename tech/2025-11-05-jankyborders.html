JankyBorders

📅 2025-Nov-05 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ macos ⬩ 📚 Archive

In MacOS Tahoe, there is no clear indication of which window is currently in focus. The only indication is that the traffic light buttons of the active window are in color, while they are greyed out for the rest of the windows.

JankyBorders is an open-source app that helps draw a colored border around the currently focused window and it solved this problem I had.

  • Install:
$ brew tap FelixKratz/formulae
$ brew install borders
  • Test the app:

To test the app from the terminal with a thick orange border:

$ borders width=10 active_color=0xFFFF7F50

Note that the first hex is the transparency, followed by RGB hex.

  • To run as service:

First create a configuration file ~/.config/borders/bordersrc:

#!/bin/bash

options=(
    width=10
    active_color=0xFFFF7F50
)

borders "${options[@]}"

Next run as service in the background:

$ brew services start borders

If you make any changes to the config file, restart the borders service:

$ brew services restart borders

Tried with: JankyBorders 1.8.4 and MacOS Tahoe 26.0.1

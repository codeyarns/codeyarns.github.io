FreeTube is a privacy-focused Youtube video watching application for Windows and Linux. It is useful to users who want to watch videos, avoid ads, keep their own list of channels/subscriptions without giving that data to Google. Because of this intentional design there will be no direct importing of your Google/Youtube account in this application. This application is similar in intention to the NewPipe app for Android.

  • Install on Windows: Download and install from: https://freetubeapp.io/#download

  • Install on Linux: It can be installed from the distribution:

$ sudo apt install freetube

However, this version can be old and missing new features. It is recommended to download and install the latest from: https://freetubeapp.io/#download

  • Subscriptions: Shows a reverse chronological feed of all the videos from all the channels you are subscribed to.

  • Channels: Shows all the channels you are subscribed to. Click on the channel to view its videos. Click on Unsubscribe to drop that channel.

  • Profiles: A profile in FreeTube is similar to a channel group in NewPipe. There is a default profile and more profile can be created, each with an unique name, color and collection of channels. To change profiles and configure them click the top-right button in the window.

  • Keyboard shortcuts: The list of supported shortcuts can be found here: https://docs.freetubeapp.io/usage/keyboard-shortcuts/

  • Export: Go to Settings -> Data Settings to find options to export subscriptions (to Freetube, YouTube or NewPipe), history and playlists.

  • Import: Go to Settings -> Data Settings to find options to import subscriptions, history and playlists.

    • Import Youtube channels/subscriptions: Go to Google Takeout and export your Youtube subscriptions. Google will make it available to you as a CSV file. Import this CSV file in Freetube as above. Once imported, you will find all the recent videos from your channels listed under Subscriptions.

  • Set default playback speed: Go to Settings -> Player Settings and set the Default Playback Rate. I like to keep mine at 1.5x. The speed can also be set per-video during playback from the video‚Äôs controls.

  • SponsorBlock: This is supported, but disabled by default. Enable and configure it under Settings -> SponsorBlock Settings. For each kind of sponsor content, you get to set a unique color (to display in the seek bar) and 3 choices: Auto Skip (FreeTube automatically skips over the content for you), Show in seek bar (color is displayed in seek bar, you can use your mouse to go past this content) and Do Nothing (no color displayed in seek bar).

Tried with: FreeTube 0.18.0 and Windows 10 x64

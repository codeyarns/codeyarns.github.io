Perplexity is one of the many new startups that offer web search powered by LLMs and DL. The AI Search extension for Firefox brings that search capability directly into the browser.

  • Install: Install the extension from here.

  • Usage: Clicking the extensionâ€™s button from the toolbar opens a sidebar where your search query to Perplexity can be entered. There does not seem to be any way to set a keyboard shortcut for this.

  • Search engine: The extension also adds Perplexity into the list of Firefox search engines. Go to the search settings to set Perplexity as the default search engine or move its position among the rest of the engines. Perplexity can also be used for searching by typing @AI in the addressbar and then type in the rest of the search query.

Tried with: AI Search 0.2 and Firefox 121.0

In GViM, the Syntax â†’ Show filetypes in menu feature is turned off by default. This is quite irritating if one is dealing with code in unnamed files or code snippets, which need to be highlighted using the right syntax. To keep this feature always turned on, add this line to your vimrc file:

let do_syntax_sel_menu=1

let is a command to set internal variables and do_syntax_sel_menu is the internal variable that corresponds to this feature.

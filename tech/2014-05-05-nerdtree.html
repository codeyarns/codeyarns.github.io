NERDTree

(First posted on: 2014-05-05 14:31:08+00:00)

[caption id="attachment_4513" align="aligncenter" width="640"][](http://codeyarns.files.wordpress.com/2014/05/20140505-nerdtree.png) NERDTree plugin for Vim[/caption]

If you miss the Solution Explorer in Visual Studio or Eclipse, then you must try the NERDTree plugin for Vim.

  • Installation is easy. I like to clone the code from here and install it using Pathogen.

  • To use NERDTree, open Vim and type :NERDTree. The NERDTree window shows all the files and directories under the current directory.

  • To open any of the files or directories shown in the NERDTree window, navigate to it using jk and press Enter. The file is opened in a window to the right.

  • To switch between the NERDTree window and the code window, I like to use Ctrl-ww.

  • To make a directory in the NERDTree window as the root, place cursor on it and press C.

  • To make hidden files and directories visible, press I.

Tried with: Vim 7.4 and Ubuntu 14.04

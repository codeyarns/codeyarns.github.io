View tree of processes on Windows

2012-10-11 06:53:42+00:00

The tlist tool can be view all the processes running on Windows displayed as a tree of parent-child processes:

$ tlist -t

The tlist tool ships in the Windows Driver Kit.

Tried with: Windows Driver Kit 7.1.0 and Windows 7 x64

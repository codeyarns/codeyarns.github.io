Share to Mastodon

📅 2023-May-29 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ firefox extension ⬩ 📚 Archive

Once I started using Mastodon to share links as posts, I felt the need to have a Firefox extension to aid the job. The Share to Mastodon extension seems to do this job perfectly.

  • It can be installed from here.
  • The source code can be found here.
  • Click the addon’s button while on a webpage to get a popup window of posting that link. All the Mastodon posting options will be available in that popup window.
  • Keyboard shortcut to do the same is Ctrl + Shift + Y.

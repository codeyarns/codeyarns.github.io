Terminal is the default terminal application on MacOS.

  • Change font size temporarily: Press command + to increase, command - to decrease and command 0 to reset the font size.

  • Change font size persistently: Go to Settings -> Profiles and change the font size for the profile you use. The font size change is not yet applied, until you right-click on the terminal window, choose Inspector and choose the profile again. The change will also be applied the next time Terminal is opened.

Tried with: Terminal 2.14 and MacOS Sequoia 15.2

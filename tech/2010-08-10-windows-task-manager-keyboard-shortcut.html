  • Ctrl+Shift+Esc: To start it. To bring it up from minimized state. To restart it if it's hung.
  • Double-click in any empty space in Performance tab: To display just that graph without any window UI. This when combined with Always on Top can be used to keep a monitor graph at a corner of your desktop.
  • File -> Run new task: To launch any program, even in administrator mode. Combined with Ctrl+Shift+Esc this can be used to launch programs even when entire Windows UI is hung.
  • Right-click and choose Open File Location: To see which executable file launched this process.
  • Right-click and choose Search Online: To search online about this executable file.

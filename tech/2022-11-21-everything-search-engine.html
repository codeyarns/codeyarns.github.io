Everything is my go-to search engine on Windows when I am trying to find a file or folder. This is a search tool written by David Carpenter that responds at lightning-fast (âš¡) speed. It builds an index of everything on all NTFS volumes on the system in the background by running a Windows service that monitors all filesystem activity.

I use this tool similar to the locate tool on Linux and I consider Everything as a must-have tool on Windows.

  • Search: Just open Everything and start typing the substring of filename or path or extension you want to search. The files and directories listed underneath are automatically pared down as you type your search string.

  • Columns:

    • Different columns can be added or removed by right-clicking on any column name at the top.
    • Drag and drop a column to change the placement of columns.
    • I also like to choose the option Size all columns to fit when a path or filename is obscured because it is too long.

Tried with: Everything 1.4.1.1022 and Windows 10 x64

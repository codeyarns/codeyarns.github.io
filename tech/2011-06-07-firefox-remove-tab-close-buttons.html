Firefox: Remove Tab Close Buttons

2011-06-07 11:48:48+00:00

Firefox 4 displays the close button on each open tab. This is a wastage of the horizontal space in the tab bar. You can close a tab anyway by using the keyboard or mouse as described in this post.

To remove the tab close button, go to about:config and set the value of browser.tabs.closeButtons to 2.

For more information on this preference see browser.tabs.closeButtons at MozillaZine.

Tried with: Firefox 4.0.1

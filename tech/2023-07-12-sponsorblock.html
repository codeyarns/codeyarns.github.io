SponsorBlock created by Ajay Ramachandran provides browser extensions and API that helps in skipping sponsor segments in Youtube.

  • Install: Install the browser extensions (Firefox here, Chrome here) or an app with SponsorBlock support like NewPipe.

  • API URL: In the Settings, if the API URL is required, then provide https://sponsor.ajay.app/api/.

  • Categories: In the Settings, choose the categories of content youâ€™d like to be skip: sponsor, intermission/intro animation, endcards/credits, interaction reminds (subscribe), unpaid/self promotion, non-music section in music, preview/recap, filler tangent/jokes and such.

  • Play video: When you play the video, the seekbar is highlighted in colors of the various categories (see above) you have chosen to skip. And the player will automatically skip over the segments of the video you wanted to.

  • Source code: The code can be found at https://github.com/ajayyy/SponsorBlock

  • Database: The database for SponsorBlock is a set of CSV files, one for each type of category to block. The database files and their mirrors can be found at https://sponsor.ajay.app/database

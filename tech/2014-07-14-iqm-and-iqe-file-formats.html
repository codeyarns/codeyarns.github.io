IQM and IQE file formats

2014-07-14 03:34:46+00:00

The Inter-Quake Model (IQM) and Inter-Quake Export (IQE) file formats are used to represent skeletal animation of a character. These formats are ASCII, humand readable and easy to understand. These are the modern alternatives to the classic MD5Mesh and MD5Anim file formats.

RipperX is a simple GUI ripper to extract an audio CD to MP3 files. I used it as the equivalent of CDex on Linux.

  • Install: In addition to the program it is important to ensure that the LAME encoder is installed:
$ sudo apt install ripperx lame

If LAME is not present, the program will rip and finish the job silently without writing any output files!

  • Usage: Insert an audio CD into the CD drive. Open the RipperX program, click the Scan button to scan and display the tracks, click the CDDB button to grab the track info and then click the Go button to rip to MP3.

  • CDDB: The old CDDB server that the app uses may not work anymore. Click the Config button and the CDDB tab and set a working CDDB URL as described here.

  • Output files: The output filename and directory format can be set by clicking the Config button and the Files tab. I like setting filename format to %a - %# - %s and directory format to %a - %y - %v.

Tried with: RipperX 2.8.0 and Ubuntu 22.04

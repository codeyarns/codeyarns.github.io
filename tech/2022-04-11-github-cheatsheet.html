Github Cheatsheet

📅 2022-Apr-11 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ github ⬩ 📚 Archive

  • List of all repos I starred: https://github.com/stars

  • For search queries in a repo:

    • URL for search of a single word “foobar”: https://github.com/SOME_USER/SOME_REPO/search?q=foobar
    • URL for search of multiple words “foobar” “nature”: https://github.com/SOME_USER/SOME_REPO/search?q=foobar+nature
    • URL for search of word with spaces “foobar wins”: https://github.com/SOME_USER/SOME_REPO/search?q=%22foobar+wins%22

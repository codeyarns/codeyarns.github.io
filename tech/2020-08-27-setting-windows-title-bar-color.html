Setting Windows title bar color

Windows 10 allows the user to set a color for the title bars of application windows. By enabling this, the title bar of the active window will have the chosen color and the title bars of inactive windows are grey. This title bar coloring difference between active and inactive windows aids me a lot when working on multiple displays where multiple windows are open.

This active title bar color can be enabled as follows:

  • Open Colors settings dialog. Pick the color you want for your theme, the same will be used for title bars.
  • Scroll down to the Show accent color on the following surfaces section and enable the Title bars and window borders option.

The title bar should take immediate effect.

Sadly, many applications do not use the Windows title bar color setting and use their own title bar color. Worse, their title bar color remains the same, no matter if that application window is active or not in focus.

Here is a list of some applications where I was able to make them switch to the native title bar colors:

  • Foxit Reader v7.2.8.1124: By default, it was using the Office-style Ribbon mode where it had set its own title bar color. Go to File -> Change Toolbar Mode and switch to Classic Toolbar Mode. It now gets a traditional Windows title bar which has the color you set.
  • Visual Studio Code v1.48.2: Go to Settings. Change the window.titleBarStyle to native.

D-Link DCM-604 Wireless Router Configuration

📅 2012-Jun-17 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ d-link, starhub, wireless router ⬩ 📚 Archive

The D-Link DCM-604 is a popular wireless router provided to StarHub customers in Singapore. This router can be configured by accessing http://192.168.0.1 in a browser. The default login and password is admin and password.

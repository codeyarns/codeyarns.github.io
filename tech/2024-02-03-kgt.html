Kateâ€™s Grammar Tool (KGT) is a swiss-army knife to handle the various BNF grammar files and visualize them as railroad diagrams.

  • Source code: KGT is open source and its source code is available at https://github.com/katef/kgt.

  • Build: Instructions to build this tool:

$ git clone git@github.com:katef/kgt.git
$ cd kgt
$ bmake -r

This produces the binary build/bin/kgt.

  • Product railroad diagrams: To produce railroad diagrams for a ABNF file in ASCII text, UTF8 or SVG formats:
$ kgt -l abnf -e rrtext < in.abnf > out.txt

$ kgt -l abnf -e rrutf8 < in.abnf > out.txt

$ kgt -l abnf -e svg    < in.abnf > out.svg

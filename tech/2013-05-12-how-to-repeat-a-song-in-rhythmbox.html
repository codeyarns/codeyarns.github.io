How to repeat a song in Rhythmbox

2013-05-12 03:57:35+00:00

One of the Foobar2000 features I miss in Rhythmbox is the ability to repeat a song over and over again. Sometimes you hit a song that you like so much, you want to listen to it over and over again.

You can get this feature in Rhythmbox by installing the Repeat One Song plugin. To do this, first add the fossfreedom PPA. Next, install the plugin using the package name rhythmbox-plugin-repeat-one-song.

Open Rhythmbox, choose Edit > Plugins and enable Repeat One Song. This adds a Repeat One Song button to the toolbar. Play the song you want to repeat and press this button to have it repeat over and over again.

Tried with: Rhythmbox 2.96 and Ubuntu 12.04 LTS

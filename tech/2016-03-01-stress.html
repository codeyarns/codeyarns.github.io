stress

2016-03-01 14:58:25+00:00

stress is a useful tool to load your computer with computation, IO, memory or disk work. This can be useful, for example, to examine how a CPU behaves when a different number of cores are loaded with computation.

  • Installing is easy, with this funny invocation:
$ sudo apt install stress
  • To load 2 CPU cores with computation work:
$ stress -c 2

Tried with: Stress 1.0.4 and Ubuntu 15.10

Surprisingly for a modern desktop OS, Ubuntu does not ship with a builtin clipboard manager. I have been so spoilt by the excellent Windows clipboard manager that I immediately felt the lack of it in Ubuntu. Thankfully, GPaste, created by Marc-Antoine Perennou, can be a decent clipboard manager replacement with a bit of configuring.

  • Installing it is easy:
$ sudo apt install gpaste

  • It can be launched by typing the name GPaste from the start menu. This launches the GUI tool, whose actual path is /usr/libexec/gpaste/gpaste-ui.

  • Once I get the GUI tool, I use the up/down keyboard shortcuts to select the copied item I want from history, press Enter to select it. The GUI tool disapppears and I can now paste it using Ctrl + v shortcut whereever I want.

  • Keyboard shortcut: The default shortcut to launch the GUI tool is Ctrl + Alt + g. However, I needed this to be Win + v since I am used to that in Windows. I found that setting Win + v in the toolâ€™s own settings or in Ubuntuâ€™s global keyboard shortcuts did not work. Finally, I created a new custom shortcut for Win + v in Ubuntuâ€™s global keyboard shortcuts to launch /usr/libexec/gpaste/gpaste-ui. This worked!

  • Plain text: Being able to paste plain text is an important feature sometimes and Windows 11 supports this. However, it does not look like GPaste has this feature.

Tried with: GPaste 3.42.6 and Ubuntu 22.04

Always on top with TurboTop

(First posted on: 2020-03-21T17:26:58.275659-07:00)

An important feature I have to come to like in Linux desktop environments is that I can easily make one or more windows stay always on top. This feature is usually built into most Linux desktop environments like KDE.

Windows API has the feature, so select applications offer this feature, for example Task Manager allows you to keep it always on top. But since the Windows desktop environment does not support this fetaure, we need an application that runs in the background or in the system tray to do this across all applications.

TurboTop is the simplest application that I could find that does this always on top feature. After you download and install it, it runs in the system tray. Just click it and choose the window you like to keep always on top.

Tried with: TurboTop 2.8.0.21 and Windows 10 v1809

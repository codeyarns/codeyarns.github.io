Exporting a Wordpress blog to Jekyll is useful if you prefer Jekyllâ€™s light static webpage generation. It is also useful if you just want to export your Wordpress blog posts to Markdown files.

  • Export your Wordpress blog using its export feature. This downloads your blog as one or more XML files.
  • Clone the Exitwp repository:
$ git clone https://github.com/thomasf/exitwp.git
  • Copy the Wordpress XML files into the wordpress-xml directory that is present inside the repository.
  • Run the converter:
$ python exitwp.py
  • After the conversion is complete, you will find your Jekyll Markdown posts in the build/jekyll/your-wordpress-blog-name/_posts directory in the repository.

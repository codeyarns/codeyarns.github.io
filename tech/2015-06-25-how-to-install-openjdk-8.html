OpenJDK 8 which provides the JRE and JDK to work with Java 8 is available from Ubuntu repositories or from a PPA depending on your Ubuntu version.

  • If your Ubuntu version does not have Java 8, then add this PPA:
$ sudo add-apt-repository ppa:openjdk-r/ppa
$ sudo apt update
  • Install JRE and JDK of OpenJDK 8:
$ sudo apt-get update
$ sudo apt install openjdk-8-jre openjdk-8-jdk
  • Switch the default Java interpreter and compiler to v8:
$ sudo update-alternatives --config java
$ sudo update-alternatives --config javac
  • There are a whole bunch of other Java tools (like jdb and javadoc) that can be switched to v8 versions, if necessary. You can list these using this command:
$ sudo update-alternatives --get-selections | grep openjdk

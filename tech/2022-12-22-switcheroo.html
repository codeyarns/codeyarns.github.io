In Windows, you can switch between open windows using Alt + Tab. However, when you have more than 2 open windows, you might find yourself pressing Alt + Tab over and over again to find the window you want to switch to. I was looking for a solution to this problem and found Switcheroo. It is a task switcher for Windows where I can switch to another window by partially typing the name of that application and pressing Enter.

  • Source code: This tool is open-source and the code can be found at: https://github.com/kvakulo/Switcheroo
  • Installer: Binaries for installation can be found at: https://github.com/kvakulo/Switcheroo/releases. The most recent release seemed to be in 2018 and I just used that.
  • Invoking: Once installed the tool can be invoked using Alt + Space. This default shortcut can be changed to whatever you wish in the Options of the tool.
  • Replace default switcher: It can also take over the Alt + Tab shortcut and that is what I preferred. This did mean that switching between a pair of windows quickly Alt + Tab-ing went out the window. I have to now do Alt + Tab + Enter.
  • Switch like magic: But the reason I got this app was to switch between multiple windows and that works exactly as advertised. I can invoke the switcher and then type the minimal substring of the application name and then Enter. It displays the candidates in a dropdown so you can see and learn how to use this. For most applications, I just need a single character, for example to switch to an open Firefox window: Alt + Tab -> f -> Enter.
  • Choose from dropdown: The tool also shows the open applications in a dropdown. So for folks who like to scroll up and down a list (using up and down keys) and choose their window, this tools helps there too.

I like this task switcher and cannot imagine working without this now. The best part is that I can switch to any open window I want with a deterministic number of keypresses, with my eyes closed in fact! ğŸ˜

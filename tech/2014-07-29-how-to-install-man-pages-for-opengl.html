How to install man pages for OpenGL

📅 2014-Jul-29 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ opengl ⬩ 📚 Archive

Believe it or not, the man pages for the OpenGL API are available for query and browsing right at your console! Just install them using this command:

$ sudo apt install opengl-4-man-doc

The same can be viewed online at OpenGL 4 Reference Pages.

Tried with: Ubuntu 14.04

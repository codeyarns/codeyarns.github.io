How to build and install cgkit from source on Ubuntu

📅 2013-Aug-24 ⬩ ✍️ Ashwin Nanjappa ⬩ 📚 Archive

cgkit (Computer Graphics Kit) is a collection of Python modules that can be useful to write computer graphics programs. Currently, cgkit does not ship with Ubuntu. But, it is available at PyPI.

However, if you decide to build it from source that is pretty easy too:

  1. Before building cgkit, I found that I needed some packages:
$ sudo apt-get install scons python-pygame

  1. Download the cgkit source file from its Sourceforge page and unzip it.

  2. Building the code is a two-step process: first the C++ library using scons and then the Python wrappers and installing them:

$ cd supportlib
$ scons
$ cd ..
$ sudo python setup.py install

Tried with: cgkit 2.0.0 and Ubuntu 12.04 LTS

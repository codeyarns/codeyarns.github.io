Useful Python packages

📅 2023-Apr-17 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ python ⬩ 📚 Archive

Python is famous for coming with batteries included, that is it ships with a lot of useful packages. Also, there are a lot of packages to do pretty much anything imaginable that can be installed using PyPI.

This post lists all the packages that I find useful and what I use them for:

  • argparse: Parse input arguments and print help.

  • art: Print text as ASCII art.

  • certifi

  • colorama: Print text with foreground and background colors.

  • datetime: Get current time, convert datetime object to a string in any format.

  • dateutil: Convert date string to datetime object.

  • joblib

  • noaa_sdk: Get NWS weather forecasts.

  • norwegianblue: Weird package name, but its eol command prints out the EOL dates table for a given library or software.

  • onnx

  • python-charmers

  • pyzipcode: Given a zipcode find its city, state, latitude, longitude, timezone and DST. Given a city and state, find all its zipcodes.

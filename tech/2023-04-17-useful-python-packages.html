Useful Python packages

📅 2023-Apr-17 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ python ⬩ 📚 Archive

Python is famous for coming with batteries included, that is it ships with a lot of useful packages. Also, there are a lot of packages to do pretty much anything imaginable that can be installed using PyPI.

This page tracks the packages that I found useful to use and also those that should probably not be used.

  • certifi
  • joblib
  • norwegianblue: Weird package name, but its eol command prints out the EOL dates table for a given library or software.
  • onnx
  • python-charmers

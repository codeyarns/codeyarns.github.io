Using list as stack in Python

üìÖ 2012-Apr-18 ‚¨© ‚úçÔ∏è Ashwin Nanjappa ‚¨© üè∑Ô∏è list, python, stack ‚¨© üìö Archive

The humble list can be used as a stack in Python. Unlike using list as queue, using list as stack is optimal.

stk = []

for i in range( 0, 5 ):
    stk.append( i ) # Push to stack

while stk: # Stack not empty
    j = stk.pop() # Pop from stack
    print( j )

# Output: 4 3 2 1 0

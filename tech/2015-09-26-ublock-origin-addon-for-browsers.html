uBlock Origin addon for browsers

uBlock Origin is a great browser addon created by Raymond Hill to aid in blocking all sorts of trackers and content in webpages. This addon is open-sourced at Github here. Though I had tried this addon in 2015, I returned to it fully only in 2022 after my favorite uMatrix addon was no longer supported. I use the addon on Firefox and so my description here is based on the UI in that browser.

  • Basic mode: In this mode the addon uses the domain/filter lists maintained online by others to block trackers and content. This is called static filtering. You can configure which lists to use for this in the Filter lists tab in the addon settings.

  • Advanced mode: Choose I am an advanced user in the Settings tab to enable this. This gives a popup UI (like in uMatrix) to selectively block content/domain either globally or just for the current site. This is called dynamic filtering and it selectively overrides the static filters.

  • Trusted sites: There are some sites where I do not want any filtering to be applied. This is toggled by pressing the power button in the basic UI. The list of trusted sites is in the Trusted sites tab in the settings for manual configuration.

Dynamic filtering

This is the popup UI dialog that appears to the left of the basic dialog when the addon‚Äôs button is clicked in the toolbar.

  • Colors: The colors on the left side indicate if that particular row is being fully allowed (green), partially blocked (yellow) or fully blocked (green).

  • Rows: Each row is a certain type of content or domain. The rows at the top before the domain rows, from all to 3rd-party frames, apply generally to those types of content.

    • Clicking to the left of the all row toggles showing/hiding all the subdomains under the domain rows. So it helps in filtering the parent domain or specific subdomains.
    • Clicking the ... on certain domain rows does the same as above for that specific parent row.
    • Among these rows I like to globally block 3rd-party, 3rd-party scripts and 3rd-party frames and then selectively enable required domains on specific websites. This is called hard blocking mode in this addon.

  • Global rules: Among the two columns, the left one is for global filtering. Each cell has two options: click on the right-side in that cell to block it (it becomes red) or click on the left-side inside that cell to noop it (it become gray).

    • Note how green option (to allow) is not available, it is by design of the creator. If you really need it, you get the green option if you press Ctrl twice.

  • Local rules: Among the two columns, the right one is for local filtering. Everything else is same as the global filtering. The + or - shows a general idea of how many files are being allowed (+) or blocked (-) already.

    • +/- is less than 10, ++/-- is less than 100, +++/--- is more than 100.

  • All the filtering done by the above process appears listed in the My rules tab in settings. These are just like the uMatrix rule lines (if you are familiar with that) and indicate 4 things L to R: domain/website where the rule applies, the domain from which content is allowed/blocked, type of content and action (either block or noop).

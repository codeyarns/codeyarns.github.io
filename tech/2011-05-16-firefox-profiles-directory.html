Firefox: Profiles Directory

(First posted on: 2011-05-16 03:32:36+00:00)

Firefox stores all the data and settings specific to a user in its Profiles directory. On Windows, this will be a directory named SomeRandomString.default found at %APPDATA%\Mozilla\Firefox\Profiles\

For more info on the Profiles directory go here.

Tried with: Firefox 4

Thunar

2016-02-11 12:30:19+00:00

Thunar is the default file manager for XFCE desktop environment. It is written in GTK and is quite fast.

It has all the features I want, except one: split or dual pane views. There is a patch going around for years, but Thunar that ships does not have it.

Tried with: Thunar 1.6.3 and Ubuntu 14.04

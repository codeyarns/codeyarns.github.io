Files can be shared over a home or local network using SMB or Windows sharing or SSH-based techniques (like SSHFS). But sometimes it is easiest to run a simple HTTP web server on the source computer and browse the files in any web browser on client computers on the network. To do this we need to run a HTTP server on the source computer.

Python HTTP Web Server

The simplest HTTP web server that allows file browsing is the one that is built into Python called http.server.

  • To serve the content in the current directory:
$ python3 -m http.server

This serves on port 8000 and you can access the content in any browser on the home network at http://<ip-address-of-server>:8000/.

  • To serve at a different port, say 10000:
$ python3 -m http.server 10000
  • To serve a directory other than the current directory:
$ python3 -m http.server --directory /path/to/dir
  • If you are still using Python 2.x:
$ python -m SimpleHTTPServer

Tried with: Ubuntu 22.04

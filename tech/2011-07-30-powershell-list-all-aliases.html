PowerShell: List All Aliases

2011-07-30 04:43:10+00:00

PowerShell comes built-in with aliases for almost all commands from the Windows command prompt (like dir) and the most common commands from Linux (like ls). You can also define your own aliases if you find the PowerShell commands too verbose for your taste.

To view the list of all aliases currently available and their definitions, type:

Get-ChildItem alias:

The easiest way to remember this is that all the PowerShell aliases live in a separate drive whose name is alias: So, to list the contents of this drive we use the above command.

Since, the aliases dir and ls are also mapped to Get-ChildItem, you can also list all the aliases by typing either of:

dir alias:
ls alias:

Pick your poison :-)

Tried with: PowerShell 2.0

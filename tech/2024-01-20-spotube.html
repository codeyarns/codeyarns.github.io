SpoTube

📅 2024-Jan-20 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ youtube ⬩ 📚 Archive

SpoTube is a music player application that combined the data API from Spotify to play songs from Youtube.

  • Source code: This is an open source application and its source code is available on Github at https://github.com/KRTirtho/spotube/.

  • Install: For Ubuntu, download the latest DEB file the releases. Install it using:

$ sudo install ./Spotube-linux-x86_64.deb

  • Login: Login into Spotify on Firefox. Press F12 to open the Developer Tools. Click on Storage -> Cookies and from the https://open.spotify.com entry copy the sp_dc and sp_key values. Choose to login in SpoTube and enter these values.

  • Usage: Once logged into SpoTube, search songs, add them to playlists and play them. Everything is effected on the actual Spotify account.

Tried with: SpoTube 3.4.0 and Ubuntu 22.04

There is no direct way to share files between a computer and an Android device. How tragic!

Using Bluetooth

  • Turn on Bluetooth on the Android device and enable it to be discovered by the Windows computer.
  • On the Windows computer, open Bluetooth, find the Android device and pair with it. You might need to confirm a set of digits that the Android device displays for you.
  • To share file from Windows to Android: In the Bluetooth settings, choose Send or receive files via Bluetooth, choose the file and send it. On my Android device, I get a prompt to download the file and it is downloaded to a bluetooth directory on the internal storage.

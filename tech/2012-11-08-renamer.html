ReNamer

(First posted on: 2012-11-08 14:13:09+00:00)

[caption id="attachment_2886" align="aligncenter" width="414"][](http://codeyarns.files.wordpress.com/2012/11/20121108-renamer.png) ReNamer[/caption]

ReNamer is an excellent utility to rename multiple files. It is far better than Bulk Rename Utility, which I was using earlier for this purpose. The user experience is much easier and straightforward. Bulk Rename Utility fills the UI with a scarily large number of operations. Instead, ReNamer uses a wizard which can be used to add many rules to rename the files. The rules are executed in the order specified and the results are previewed instantly. If the preview looks fine, press Rename and the changes to the filenames are made permanent.

Tried with: ReNamer 5.6 and Windows 8 Pro x64

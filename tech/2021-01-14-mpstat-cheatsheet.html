mpstat Cheatsheet

📅 2021-Jan-14 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ mpstat ⬩ 📚 Archive

The mpstat tool allows you to query and monitor statistics related to processors and its cores.

  • Installing the tool is easy:
$ sudo apt install sysstat
  • To view stats on all the cores on your computer:
$ mpstat -P ALL

$ mpstat -P ALL -o JSON

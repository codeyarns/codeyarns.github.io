Vimium

📅 2022-Dec-22 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ chrome, firefox, vim ⬩ 📚 Archive

Vimium is a great browser extension for Vim fans who want to operate their browser and webpages just like they do in Vim.

  • Source code is at: https://github.com/philc/vimium
  • Chrome extension can be installed from here.
  • Firefox extension is called Vimium-FF and can be installed from here.

Keyboard shortcuts

  • ?: Display all the keyboard shortcuts.
  • Tabs
    • o: Open a webpage in the current tab. Start typing search terms (for Google search) or page title to load from your browser history. The search engine can be changed (to DuckDuckGo for example) in the Advanced Options of the addon.
    • O: Same as o, but open in a new tab.
    • x: Close current tab.
    • X: Restore closed tab.
    • H: Go back to previous page from history.
    • L: Go forward to next page from history.
    • T: Jump to any open tab. It shows a dropdown list of all open tabs. Start typing page title to get the tab you want to switch to and press Enter.
  • Navigation inside webpage
    • j or k: Scroll down or up.
    • d or u: Scroll half-page down or up.
    • Clicking a link on the webpage
      • Press f and you will see 1 or 2 character names displayed beside each link. Type the characters shown beside the link you want. This opens the link in the current tab.
      • Use F instead of f to open in a new tab.
    • /: Type the string to find in webpage and press Enter.
    • n or N: Jump to next or previous string in the results from / search.

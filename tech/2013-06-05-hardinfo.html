HardInfo

📅 2013-Jun-05 ⬩ 📚 Archive

There are tools like PC Wizard on Windows that shows all the hardware details of a computer. HardInfo is a similar tool for Linux.

  • It can be installed easily from the Ubuntu repositories:
$ sudo apt install hardinfo

  • It can started as hardinfo from the shell or as System Profiler and Benchmark in the Ubuntu Launcher.

  • Besides providing system information, it also has some simple CPU and FPU benchmark tools.

  • A report can be generated for any computer and saved in TXT or HTML format. This can be useful to compare different computers.

Tried with: HardInfo 0.5.1 and Ubuntu 14.04

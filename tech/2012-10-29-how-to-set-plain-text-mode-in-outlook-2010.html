How to set plain text mode in Outlook 2010

2012-10-29 09:46:55+00:00

I like to use plain text mode by default for composing and replying to emails in Outlook 2010. To do this, go to File → Options → Mail. In the Compose Messages section, choose Plain Text from the dropdown menu for Compose messages in this format.

Finestra Virtual Desktops

Ashwin Nanjappa

2011-05-27 11:43:28+00:00

Finestra Virtual Desktops is yet another contender for the virtual desktop manager crown on Windows. I tried it to see if it can fix a problem I have with Dexpot: a window in an inactive desktop which throws up an alert shows up on my active desktop. Finestra Virtual Desktops has an option to make windows sticky, jailing them inside their enclosing desktop. It fixes this, but I found Finestra to be a work-in-progress, there are a lot of small bugs all over the place. The most irritating for me was that windows on a desktop would rearrange if I switched away and back to it. Due to these reasons, Finestra is not usable for me right now.

Tried with: Finestra Virtual Desktops 1.2 and Windows 7

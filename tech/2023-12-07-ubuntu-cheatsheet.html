Ubuntu Cheatsheet

📅 2023-Dec-07 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ cheatsheet, ubuntu ⬩ 📚 Archive

This post is an attempt to gather together the tricks and tweaks that make my life easier across the many computers running Ubuntu that I use.

  • Insert emoji: Press Win + ., you get an underlined e character. Now type the emoji word (like smile) and press Ctrl + Space to insert the emoji character. Press Ctrl + Space again to get the other choices of emoji for the word you typed. Another alternative is to just press Ctrl + Space after you see the underlined e - this brings up the emoji picker UI. All of this can be controlled by opening the ibus-setup application. Frankly, I think all of this emoji insertion UX is terrible and non-intuitive.

  • Change mouse cursor size: I like to have a larger mouse cursor so that it is easier to spot on the large display I use. Surprisingly, this customization is not provided in the Mouse section of the Settings. Instead, go to Settings -> Accessibility. Click on Cursor Size to change it from the default to any of the 4 larger sizes provided. In this same section, I also like to enable the Locate Pointer option that animates a bullseye on the cursor when the Ctrl key is pressed.

  • Change mouse cursor color: The mouse color is controlled by the desktop theme. Install GNOME Tweaks (gnome-tweaks) and open it. Go to Appearance -> Themes -> Cursor and choose a different cursor theme. The default Yaru and DMZ-Black makes the cursor black, DMZ-White/Whiteglass makes it white, Redglass makes the cursor red, and Handhelds turns it into a tiny cursor.

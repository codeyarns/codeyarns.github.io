wget cheatsheet

2016-01-22 12:49:11+00:00

  • To download a URL:
$ wget url_to_download

It writes to the current directory with a filename same as in the URL.

  • To download a URL, but write to a specific filename:
$ wget -O write_to_this_file url_to_download
  • To download a URL and write inside a specific directory:
$ wget -P this_dir url_to_download
  • To download a list of URLs stored in a file, one URL per line:
$ wget -i list_of_urls.txt
  • By default, wget is quite verbose and displays progress bar and other details for each URL. To download quietly:
$ wget -q url_to_download

Tried with: Wget 1.15 and Ubuntu 14.04

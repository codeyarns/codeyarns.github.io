No libglog ImportError

📅 2017-Oct-12 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ glog, importerror ⬩ 📚 Archive

Problem

Installing a package with Pip gave this error:

ImportError: libglog.so.0: cannot open shared object file: No such file or directory

Solution

Install the Google Log package:

$ sudo apt install libgoogle-glog-dev

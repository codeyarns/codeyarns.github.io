Windows 11 has added increasingly invasive features that track and send back lots of information about the user. It is hard to go through all the settings and registry to individually disable them. The O&O ShutUp10++ tool makes it very easy to control these.

  • Download this free tool from here. Run the OOSU10.exe file you downloaded. It requires no installation.

  • You can either individually look at all the knobs and decide to change them. Or what I like to do is to choose Actions -> Apply only recommended settings. Close the app after this and you may need to restart Windows for it to take effect.

  • Windows 11 continues to toggle back or add such tracking features in every update. So it makes sense to download this tool every few months and run it again.

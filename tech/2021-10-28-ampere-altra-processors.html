Ampere NUMA Control (ANC)

This is an option in the BIOS of Gigabyte systems to configure NUMA for the Altra processors. There are 3 settings to choose from:

  • Monolithic: Each physical chip is a NUMA node.
  • Hemispheres: Each physical chip is 2 NUMA nodes.
  • Quadrants: Each physical chip is 4 NUMA nodes.

LaTeX Error: File bbm.sty not found

(First posted on: 2014-05-25 04:14:03+00:00)

Problem

Compiling LaTeX file gives this error:

File bbm.sty not found

Solution

It requires installation of this package:

$ sudo apt install texlive-fonts-extra

Tried with: Ubuntu 14.04

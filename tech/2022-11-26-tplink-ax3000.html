The TP-Link AX3000 (also called TP-Link Archer AX55) is a dual band gigabit wifi6 router. I decided to get an updated and more powerful wifi router at home after being disappointed with the signal strength of my TP-Link Archer C1200 wifi router and the TP-Link TL-WPA8631P powerline wifi extender.

  • Physical setup: The setup was pretty straightforward - power and LAN input from the broadband modem. The internet LED light was still orange - it turned green once I rebooted the broadband modem.

  • Connecting to wifi the first time: From my computer I connected to the TP-Link_XXXX SSID network and password listed under the modem. Once connected, all the configuration can be done by connecting to http://<your-router-ip> or http://tplinkwifi.net/ in the browser.

  • Change settings: Some of the settings I changed were to rename the SSID network names and change the IP address pool.

Specifications

  • Wifi
    • IEEE 802.11ax/ac/n/a 5 GHz 2402 Mbps
    • IEEE 802.11ax/n/b/g 2.4 GHz 574 Mbps
  • CPU: Intel AnyWAN GRX350
  • Wifi chipset: Intel WAV654

Software

  • Hardware Version: Archer AX55 v1.0
  • Firmware Version
    • 2022/11/25: 1.1.1 Build 20220621 rel.71166(5553)

Usage

  • Reboot schedule: If you find that rebooting the modem every day or week is useful, it can be scheduled to do that automatically. Go to System -> Reboot -> Reboot Schedule to set the time and cadence.

  • Eco Mode: The modem can be set to run in Power-Saving or Ultra Power-Saving modes for certain hours of the night. Go to System -> ECO Mode to set this.

  • Traffic statistics: The software of this modem does not seem to have the feature of collecting and displaying internet usage and other traffic statistics.

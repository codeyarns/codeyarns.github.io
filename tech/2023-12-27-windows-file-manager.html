The classic old Windows File Manager shipped in Windows 3.0, with the icon of an actual file cabinet. This is now open sourced and available for install on Windows 11.

  • Install: The easiest method is to install from the Windows store here.

  • Source code: The source code can be found on Github here.

  • What I like:

    • Simple UI and very fast!
    • Density of information is high. I can see lots of the files and directories in a single screen compared to modern File Exlorer.
    • Does not talk to the internet for anything. This also contributes to its speed.
    • Supports multiple windows inside the main windows. These can be tiled or arranged in different ways.

  • What I donâ€™t like:

    • No way to bookmark important directories.
    • Does not show the smart user directories like Documents, Downloads etc.

Tried with: Windows File Manager 10.2.0.0 and Windows 11 x64

PDFSeparate is a tool that can be used to pull out the individual pages of a PDF file, each page into a separate PDF output file.

  • To install this tool, install the poppler-utils package which it ships under:
$ sudo apt install poppler-utils
  • To separate every page in in.pdf into separate pages of the form out-1.pdf, out-2.pdf and so on:
$ pdfseparate in.pdf out-%d.pdf

Tried with: Poppler 0.62.0 and Ubuntu 18.04

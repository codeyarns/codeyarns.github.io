Disable add-ins in Outlook 2010

📅 2012-Oct-23 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ outlook ⬩ 📚 Archive

Problem

Outlook 2010 can take a while to start because it is loading many add-ins. These add-ins may have been enabled by the system administrator at the workplace by default.

Solution

Turn off all the add-ins that are not really necessary. Go to File → Options → Add-Ins. Click the Go button beside the Manage COM Add-ins dropdown. In the COM Add-Ins dialog uncheck all the add-ins you do not need. Press OK, close and start Outlook 2010. It should load up faster.

Tried with: Windows 7 Enterprise x64

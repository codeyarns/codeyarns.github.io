Every now and then I find myself needing to rename a lot of files generated from different sources. On Windows, I used to use tools like ReNamer and Bulk Rename Utility for this purpose. I found a good alternative for Linux in pyRenamer.

pyRenamer can be installed using the package name pyrenamer:

$ sudo apt install pyrenamer

It can be invoked from the Ubuntu Launcher using pyRenamer.

Tried with: pyRenamer 0.6.0 and Ubuntu 12.04 LTS

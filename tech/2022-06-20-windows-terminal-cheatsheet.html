Windows Terminal Cheatsheet

📅 2022-Jun-20 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ cheatsheet, windows terminal ⬩ 📚 Archive

  • Ctrl + v cannot be used for visual blocking in Vim since it is bound to paste by default. Go to Settings -> Actions and delete that binding. There is still Ctrl + Shift + v for pasting into the terminal.

Tried with: Windows Terminal 1.13.11431.0

How to mount and unmount on WSL

📅 2020-Jun-17 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ mount, wsl ⬩ 📚 Archive

Any drive of any filesystem that is already mounted under Windows can be then mounted under WSL. What makes this possible is a filesystem plugin in WSL named DrvFS.

  • The C: drive is mounted by default under WSL at /mnt/c.

  • To mount a different drive under WSL, create a directory under /mnt and mount using the DrvFS plugin using this syntax:

$ sudo mkdir /mnt/k
$ sudo mount -t drvfs K: /mnt/k
  • You can see the list of mounted drives and their mount options using:
$ cat /etc/mtab
  • To unmount a mounted drive:
$ sudo umount /mnt/k

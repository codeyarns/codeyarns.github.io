Advanced search

Twitter has an excellent advanced search option that can be used to find old tweets made by yourself or others. Access it here: twitter.com/search-advanced

Here are the various terms that can be used in the query bar at the top:

  • from:nytimes: Only show the popular tweets from this account.
  • git mercurial svn (from:nytimes) since:2018-12-01: Show the tweets by nytimes with any of the terms git mecurial svn starting from date 2018-12-01.
  • "git mercurial svn": Search exactly for the full string "git mercurial svn" in tweets.
  • until:2009-11-13: Search tweets until this date.
  • +filter:nativeretweets: Search just retweets.
  • -filter:replies: Do not include replies (which are included by default otherwise).

When Twitter shows these results, make sure to click on the Latest tab to see all results. Else Twitter will only share the top ones by default.

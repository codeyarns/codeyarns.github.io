Navigate Up WE extension for Firefox

📅 2023-Dec-22 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ firefox extension ⬩ 📚 Archive

Navigate Up WE is an extension for Firefox that makes it easy to move up the levels of the URL path.

  • Install: It can be installed from here.

  • Usage: Click the button of the extension or use the keyboard shortcut Alt + u to move up the URL.

Tried with: Navigate Up WE 4.4, Firefox 119.0.1 and Ubuntu 22.04

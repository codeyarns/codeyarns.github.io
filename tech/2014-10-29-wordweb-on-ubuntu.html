I have used the WordWeb free dictionary as my primary dictionary for many years, on both Windows and Android. When I use Linux, I immediately miss having a Linux version of WordWeb.

Thankfully, it seems to run fine under Wine as described below:

  • Install Wine, if you do not have it already:
$ sudo apt install wine

  • Download the Windows installer of WordWeb from here.

  • Run the WordWeb installer using Wine:

$ wine wordweb10.exe
  • Wine fully installs the app so that I can launch it using its name WordWeb in the desktop and lookup words. It can even be minimized to the system tray of GNOME/Ubuntu desktop!

Tried with: WordWeb 10.33, Wine 6.0.3 and Ubuntu 22.04

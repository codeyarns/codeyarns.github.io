WordWeb on Ubuntu

(First posted on: 2014-10-29 12:46:04+00:00)

I use the WordWeb free dictionary application, both on Windows and Android. It does not ship a program for Linux. Thankfully, WordWeb works fine under Wine.

  • Install Wine, if you do not have it:
$ sudo apt install wine

  • Download the Windows installer of WordWeb from here.

  • Right-click on the downloaded EXE file and choose to install using Wine. The installation should work fine and you end up having the Wordweb icon in the bar at the top. Click it to use WordWeb.

Tried with: WordWeb 7.1, Wine 1.6.2 and Ubuntu 14.04

How to insert SVG image in IPE

Ashwin Nanjappa

2014-06-02 07:15:23+00:00

To insert a SVG image into IPE, it first needs to be converted to IPE format. The svgtoipe program can be used to do this.

  • First, install this program and convert the image from SVG to IPE format:
$ sudo apt install svgtoipe
$ svgtoipe foo.svg foo.ipe
  • Next, open the IPE image file and copy it to the IPE session you are working on as explained here.

Tried with: IPE 7.1.4 and Ubuntu 14.04

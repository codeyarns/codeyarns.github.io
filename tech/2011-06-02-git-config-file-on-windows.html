Git: Config File on Windows

(First posted on: 2011-06-02 13:41:10+00:00)

On Windows, Git stores its configuration file .gitconfig in the directory %HOMEDRIVE%%HOMEPATH%.

For a user named Joe on Windows 7, his .gitconfig will typically be found in C:\Users\Joe

Tried with: Git 1.7.4 and Windows 7

PuTTY is a popular SSH client for Windows. It was created by and is maintained by Simon Tatham.

  • Install: Download and install the latest version of PuTTY from here.

  • Help: There are a lot of configuration options in the PuTTY launch window. To get detailed help about all these options, click the ? at the top-right of the window and click on the option you want help for.

  • Sessions: Configuration related to a host can be stored together in PuTTY as a session. However, the UI for this is a bit non-intuitive, here is a primer:

    • To create a session: type a name in the Saved Sessions textbox, provide its hostname, port and connection type. Click the Save button to save this session.
    • To load a session: select its name in the session list and press the Load button. This loads the configuration values of this session into all the categories listed in the sidebar on the left.
    • To configure a session: load the session, click through the categories in the left sidebar and set the values you desire. Remember to go back to the Session category and click the Save button to save this configuration.
    • To open a session: choose the session in the list and either double-click it or click the Open button. This opens a SSH connection with the host of that session.

  • Cursor: Find this configuration under Window -> Appearance. I like the vertical line with the cursor blinks enabled.

  • Font: Find this configuration under Window -> Appearance.

Tried with: PuTTY 0.80

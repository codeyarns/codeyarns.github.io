Firefox: New Tab and Close Tab

Ashwin Nanjappa

2011-05-31 03:12:35+00:00

The New Tab button in Firefox is one of those button you can do without. The keyboard shortcut to create a new tab is Ctrl+T. If you prefer the mouse, middle-click in any empty space in the tab bar to create a new tab.

To close the currently active tab, use Ctrl+W. If you prefer the mouse, middle-click on any tab to close it.

Tried with: Firefox 4.0.1

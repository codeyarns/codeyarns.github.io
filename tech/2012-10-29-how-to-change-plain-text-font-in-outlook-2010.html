How to change plain text font in Outlook 2010

(First posted on: 2012-10-29 10:11:41+00:00)

Outlook 2010 uses a horrible looking font by default for plain text. To change the font used for plain text: go to File → Options → Mail. Click on Stationery and Fonts and set the font you want in Composing and reading plain text messages section.

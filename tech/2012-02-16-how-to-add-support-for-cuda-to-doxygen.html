How to add support for CUDA to Doxygen

2012-02-16 04:55:12+00:00

Doxygen does not generate documentation for CUDA source files by default. The screenshot above shows how to enable Doxywizard to generate for .cu files. In the Expert tab, Input topic, find the FILE_PATTERNS section. Type *.cu and press the + button to add.

If you want to edit the Doxyfile directly, the steps are similar. Search for FILE_PATTERNS in the file and add *.cu to its list.

To get started quickly, I have shared my Doxyfile for C++ and CUDA code here. Just look for foobar in the file and replace with your project name and directories.

Tried with: Doxygen 1.7.6.1

How to page the output of any command at the Fish shell

Ashwin Nanjappa

2013-06-21 13:49:01+00:00

The Fish shell has a convenient shortcut to pass the output of any command to a pager like less. The default keyboard shortcut to do this is Alt-P. That is, type the command and press Alt-P. This adds a | less; at the end of the command.

Tried with: Fish 2.0.0 and Ubuntu 12.04 LTS

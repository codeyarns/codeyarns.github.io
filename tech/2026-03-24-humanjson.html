human.json

📅 2026-Mar-24 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ web ⬩ 📚 Archive

human.json is a simple decentralized mechanism created by Beto Dealmeida to discover human-authored websites and for owners of such sites to declare authorship. It essentially helps build a human-curated network of trusted sites.

For website owner

To assert that your website is human-authored, follow 2 simple steps:

  • Add a human.json at the root of your domain in this format:
{
    "url": "https://codeyarns.com",
    "version": "0.1.1",
    "vouches": [
        {"url": "https://nedbatchelder.com", "vouched_at": "2026-03-22"},
        {"url": "https://daringfireball.net", "vouched_at": "2026-03-22"},
        {"url": "https://stratechery.com", "vouched_at": "2026-03-23"}
    ]
}

The contents of the JSON file describe the domain of your own site and also list the other websites which you believe are human-authored.

  • Add this line in the header of your HTML files:
<link rel="human-json" href="/human.json">

For web users

  • Install the Firefox extension from here. Make sure the addon is pinned to the toolbar, so that its traffic light icon is visible.

  • Toolbar icon is ⚪️ when there is no information on the website you are visiting.

  • Toolbar icon is 🟢 or 🟡 or 🟠 depending on whether it is 1/2/3 hops away from a website you trust as human-authored.

  • When you visit a website which has human.json, you can click on the addon button and click Trust this site to add it and the list of sites in its human.json file to your network (being tracked by your addon).

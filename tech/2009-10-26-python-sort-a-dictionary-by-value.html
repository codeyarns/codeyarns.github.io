Python: Sort a dictionary by value

2009-10-26 07:16:28+00:00

Also see: http://stackoverflow.com/questions/613183

adict = {"nine": 9, "one":1, "eight":8, "three":3}
import operator
sorted(adict.items(), key=operator.itemgetter(1)) # [('one', 1), ('three', 3), ('eight', 8), ('nine', 9)]

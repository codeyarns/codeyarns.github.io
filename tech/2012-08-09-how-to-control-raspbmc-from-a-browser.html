How to control Raspbmc from a browser

(First posted on: 2012-08-09 15:19:51+00:00)

[caption id="attachment_2715" align="aligncenter" width="640"][](http://codeyarns.files.wordpress.com/2012/08/20120809-raspbmc-browser.png) Remote control your Raspbmc from a browser.[/caption]

I assume you are using your Raspberry Pi as a HTPC by running Raspbmc on it. (Here is how to do that.) I also assume the Raspberry Pi is connected to your home wireless router or home network. Raspbmc can easily be controlled from any other computer on this home network.

  1. In your Raspbmc, navigate to System ‚Üí System info ‚Üí Summary. Take note of the IP address (for example: 192.168.0.12) that has been assigned to your Raspbmc.

  2. Navigate to System ‚Üí Settings ‚Üí Services ‚Üí Webserver.

  3. Enable the option Allow control of XBMC via HTTP. Take note of the port (for example: 8080), username (for example: xbmc) and password (for example: no password).

  4. On any computer connected to your home network check if you can ping the IP address of your Raspbmc. For example, ping 192.168.0.12. You should be able to ping it, else check the settings of your wireless router.

  5. Open a browser on the computer and access http://192.168.0.12:8080¬†The browser will ask you for the username and password. You should be able to control XBMC from the menu and by clicking the buttons on the klunky old remote control shown in the browser.

Tried with: Raspberry Pi Model B and Raspbmc RC3

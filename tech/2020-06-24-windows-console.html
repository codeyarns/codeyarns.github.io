Windows Console

📅 2020-Jun-24 ⬩ ✍️ Ashwin Nanjappa ⬩ 📚 Archive

Windows Console is the default terminal emulator used in Windows for its command prompt/shell, for Powershell and for WSL. A replacement called Windows Terminal with modern and Linux-terminal-like features is available for newer versions of Windows 10. However, Windows Console is likely to remain in Windows for eternity due to backwards compatibility.

  • Copy: Mark the text in the console with mouse. The marked text should be highlighted. Press Enter to copy.

  • Paste (using mouse): Click top-left corner of window, choose Edit -> Paste.

  • Paste (using keyboard): Pasting is pretty common and using a mouse for that is onerous, especially because it involves going through a menu. Thankfully, you can now paste using Ctrl + Shift + V. To enable this go to Properties -> Options and enable the option Use Ctrl + Shift + C/V as Copy/Paste.

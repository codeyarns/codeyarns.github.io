SMD file format

📅 2014-Jul-14 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ smd ⬩ 📚 Archive

The StudioMDL Data (SMD) file format is used to represent skeletal animation of a character in a game. It is used to store a skeleton, its mesh and its animation per frame.

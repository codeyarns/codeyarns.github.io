How to search and replace only full word in Vim

2015-02-12 12:10:14+00:00

By default, the search and replace in Vim replaces the search string even if it is a substring in a longer string. For example, :%s/foo/abracadabra/ will replace foo that is in foobar[10] or zoofoomoo(20). However, while working with source code we typically want to replace only those occurrences where the search string appears as a whole word.

Microsoft Word Cheatsheet

Microsoft Word is the popular word processor software from the Microsoft Office suite.

  • The Windows desktop application is called Word for Windows. The Word that can be used in the browser is called Word for the web. It is a lightweight version of the Word for Windows and lacks many of the features that are in the Windows version.

  • Word also appears as part of the Microsoft 365 suite, which is a subscription based collection of Office tools, Sharepoint, Teams and others.

  • Microsoft‚Äôs online documentation for Word is extensive. However, most of the customizations and operations listed in the documentation are not possible in Word for the web. You may need to open the Desktop app to get those tasks done.

  • Change default font:

    • There does not seem to be a way to change the default font in Word for the web. The info below is for Word for Windows.
    • The default font applied in your document is dependent on what style is currently selected. The default style is Normal, but there are other styles like Heading 1 that are applied depending on the current context of the text. So, to change the default font we need to set the font for all the styles we are interested in.
    • To change a particular style, open the Styles dialog or press Ctrl + Alt + Shift + s. Hover over the style you want to change, click the dropdown arrow and choose Modify. In the dialog box, set the font, size and other parameters, choose Automatically update and New documents based on this template options.

  • Zoom: Press Ctrl + and Ctrl - to zoom in and zoom out of the doc. Press Ctrl 0 to return the doc to the default 100% zoom.

  • Split view: Press Ctrl Alt s to split and unsplit the current doc window. Split view can be useful to refer to a different section of the doc while keeping focus at the current position.

  • View current style: The style at the cursor position can be seen highlighted in the dropdown list of styles at Home -> Styles. I find it easier to view the current style highlighted in the Styles window that appears on pressing Ctrl Alt Shift s.

  • View formatting characters: Press Ctrl Shift 8 to view all the formatting characters like tab, paragraph end and other markers displayed amidst the text.

  • Reveal formatting: Press Shift F1 to bring up the Reveal Formatting sidebar that shows the full formatting details of the text at the current cursor position.

  • Find in hyperlinks: Find of text does not look inside the hyperlinks in the document. To be able to find in links, expose the links by pressing Alt + F9. Pressing it again hides the links.

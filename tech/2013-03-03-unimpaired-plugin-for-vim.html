unimpaired is yet another useful Vim plugin by Tim Pope. It provides convenient key sequences for common operations in Vim.

Some of the mappings from unimpaired that I find most useful:

  • ]b and [b to switch forward and backward between buffers

  • ]space to add a new line without going into Insert mode

  • ]q and [q for :cnext and :cprev

Tried with: Vim 7.3

iKBC CD87 BT Keyboard

📅 2022-Jan-06 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ bluetooth, keyboard ⬩ 📚 Archive

Sometime before COVID work from home began, I was thinking about ways to reduce the wires at my desk. I switched from a wired trackball to a wireless trackball, and the next target was my wired mechanical keyboard. Back then there were few options for wireless mechanical keyboards and I chose the iKBC CD87 BT keyboard in the March of 2020.

This is an economical tenkeyless mechanical keyboard which uses Bluetooth for the wireless connection. It uses Cherry MX keys (yet another reason why I wanted it) and I got the one with Cherry MX Blue keys (I like tactile and audible feedback). This keyboard is created by a Taiwanese company, but I was able to order it from Amazon US and it was delivered with no hassles. Since it uses Bluetooth, no dongle needs to be attached to the computer and it also comes with a wired USB connection too, just in case.

The setup instructions and all the special Fn key combinations came printed on a small page. The keyboard has 4 dip switches underneath, which can be toggled to change some settings.

Bluetooth mode

To use the keyboard in wireless Bluetooth mode:

  • Add 2x AAA batteries at the bottom of the keyboard.

  • There is an on/off switch at the bottom, turn it to ON.

  • The keyboard can be paired with up to 4 devices. To put the keyboard into pairing mode for the first device, press and hold Fn + F1 for 3 seconds.

  • The keyboard should appear in your computer’s Bluetooth devices list as ikbc 1.

  • Choose to connect that device and you will be asked to type a pin code and press Enter on the keyboard.

  • If successful, the keyboard is now paired to the computer.

I used this keyboard in wireless mode enthusiastically for a few months, but was ultimately disappointed due to these reasons:

  • There was definitely some extra latency in wireless mode compared to any other wired keyboard I had used earlier.

  • Even worse, a few times a day the latency would explode and I would get repeating keys or stuttering keys as if the keyboard slept for a wink. This was simply unforgiveable in my eyes.

  • Adding to this mess was the battery life. About once a month, the keyboard would die right in the middle of my work, forcing me to immediately swap batteries. In comparison, my wireless trackball lasts half a year or more on the same 2 AAA batteries.

  • Since the keyboard does not provide any Bluetooth dongle to plug into a USB port on the computer, the keyboard is not usable at the BIOS setup or GRUB menu. It cannot be used until a OS with which the keyboard has been paired is operational.

USB mode

Disillusioned by the wireless mode, but still in love with the Cherry MX Blue switches, I decided to give wired mode a shot and I am happy with that.

The wired mode works, no matter if the on/off switch is off, and doesn’t use the batteries. In this mode, it is a good (though not exceptional) keyboard. There are routes underneath to guide the USB cable through center, or left or right making the cable management on the desk a tad tidier.

This keyboard has a lot of heft and weight and I like that while I use it. It is also very minimal and black and that is nice.

When connected over USB, it appears with Vendor ID as 0x060B (Solid Year Co., LTD.) and Product ID as 0x7A0D.

Cheatsheet

  • Fn + Ins: Switch to QWERTY layout.

  • Fn + Home: Switch to DVORAK layout.

  • Fn + PgUp: Switch to COLEMARK layout.

  • Fn + R: Press and hold this until you see the two lights (Caps and Scroll BT) flash 3 times. This restores factory settings. Note: The R here depends on the keyboard layout currently set (see above). I find it best to put it to QWERTY layout first, so that the R key I am pressing is actually R in both the layout and the actual keycap.

  • Fn + F1: Pair with device 1.

  • Fn + F2: Pair with device 2.

  • Fn + F3: Pair with device 3.

  • Fn + F4: Pair with device 4.

  • Put the dip switches to 1: DOWN, 2:DOWN, 3:DOWN, 4:DOWN to get the default settings.

Firmware update

In 2026, I discovered the Firmware Update page of iKBC. I downloaded the v114 version of the firmware for my keyboard. This was a CD87_20201204_V114_C.exe file so I had to switch to Windows to use it.

I connected the keyboard using its USB cable to a Windows computer and run the downloaded file. It showed up a dialog in Chinese and I pressed the left button. It next showed that my current version was 1.01 and said it would update to 1.14. I OKed and it finished the task.

When I checked the version on MacOS, the new version showed as 1.1.4 and so the old version might have been 1.0.1.

Lessons learnt

If I were to buy a wireless mechanical keyboard the next time, I think I would still want one with Cherry MX keys, but one that uses a dedicated 2.4G wireless receiver and not Bluetooth. I have seen enough problems with Bluetooth mice, trackball, keyboards and headphones, none of which occurs with wireless devices that use their own 2.4G wireless receivers. I think the pain of adding yet another receiver dongle to the computer is far lighter than the pain of pathetic lags and sleeps with Bluetooth.

The simple and versatile Sumatra PDF has been gaining useful features with every new version. It can now be used as an ebook reader too, capable of handling the EPUB and Mobi formats.

After comparing it with the Calibre ebook reader, I actually prefer Sumatra PDF for reading ebooks. The Calibre sets of tools are many and heavy. Its ebook reader has a crappy interface and the display is not that great either. In contrast, Sumatra PDF gives a dead-simple interface that put the content upfront and all one needs to do is flip to the next or previous page.

How to Find in ebook

You will that the Find option is not available when viewing a ebook in Sumatra PDF. This is because it switches to a different UI for ebooks where this option is not available.

To get back the Find option, we need to request Sumatra PDF to use the normal UI for ebooks too. To do this:

  • Go to Settings -> Advanced Options. This opens the config file %USERPROFILE%\AppData\Local\SumatraPDF\SumatraPDF-settings.txt in your favorite editor.
  • Find the EbookUI section and change the UseFixedPageUI = false option there to UseFixedPageUI = true.
  • Close Sumatra PDF and reopen it. You should see the Find option at the top in the toolbar now.

Tried with: Sumatra PDF 3.2

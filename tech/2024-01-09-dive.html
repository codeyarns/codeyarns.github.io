Dive is a visual tool to explore the layers and contents of a Docker image.

  • Install:

The installation instructions for Ubuntu from the Github README is:

$ export DIVE_VERSION=$(curl -sL "https://api.github.com/repos/wagoodman/dive/releases/latest" | grep '"tag_name":' | sed -E 's/.*"v([^"]+)".*/\1/')
$ curl -OL https://github.com/wagoodman/dive/releases/download/v${DIVE_VERSION}/dive_${DIVE_VERSION}_linux_amd64.deb
$ sudo apt install ./dive_${DIVE_VERSION}_linux_amd64.deb
  • Usage: Invoke on any image name that you have already pulled to your computer. For example:
$ docker pull alpine

$ dive alpine
  • Layers are displayed on the left. Use arrow keys to move up and down.
  • Contents of the selected layer are displayed on the right. Use arrow keys to move, Space to collapse or expand a directory.
  • Use Tab to switch between layers and contents views.

Tried with: Dive 0.11.0, Docker 25.0.5 and Ubuntu 22.04

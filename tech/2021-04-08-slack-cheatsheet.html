Slack Cheatsheet

📅 2021-Apr-08 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ cheatsheet, slack ⬩ 📚 Archive

Keyboard shortcuts

  • Ctrl /: View list of keyboard shortcuts.
  • Ctrl K: View unread channels/people in dropdown. Type people/channel name to autocomplete and jump to that person/channel.
  • Ctrl Shift K: View all recent DMs. Type multiple people names for group chat.
  • Ctrl Shift \: Add emoji reaction.
  • ↑ or ↓: Browse through messages, moving to previous or next message.
  • → or ←: Switch to Threads view and back.
  • A: Add or remove message from Saved items.
  • Ctrl Shift S: View Saved items.
  • Ctrl + or Ctrl -: Increase or decrease font size.
  • Ctrl Shift D: Hide or show left side bar.
  • Alt ← or Alt →: Step through history of all people/channels you last visited.
  • F6: Switch focus between UI sections.

User interface

  • Click the 🕒 icon at the top to see the History of all the channels you have just visited.
  • Create Sections to organize channels/people.
  • In each section, choose to view only unread and by priority.

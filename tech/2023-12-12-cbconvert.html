CbConvert

📅 2023-Dec-12 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ cbz ⬩ 📚 Archive

Comic book CBZ/CBR files can be quite large when the scans are of a high DPI. CbConvert is a tool created by Milan Nikolic that can be used to reduce the filesize by resizing or changing the image format of the files inside the CBZ. It can also be used to apply other operations like brightness changes to the image files.

  • Download and install: Download the latest Linux x64 archive from here and unzip it.

  • Usage: Run the cbconvert-gui program to open the GUI. Click the Add Files button to pick the input files. In the Output tab specify the output directory. In the Image tab specify the output image resolution and other settings. Click the Convert button to convert and output the file.

Tried with: CbConvert 1.0.0 and Ubuntu 22.04

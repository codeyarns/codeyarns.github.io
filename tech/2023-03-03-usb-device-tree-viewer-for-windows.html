The USB Device Tree Viewer,.also called USBTreeView, is a Windows tool that can be used to view the tree of USB hubs, roots and devices and their information. It is based on the source code of the USBView tool, so it looks pretty much the same.

Some of the nice features in this tool:

  • Shows the high-speed (H), full-speed (F) and super-speed (S) designation on the tree icons itself.
  • Shows the actual name of the device (and not just the device type) in the tree view itself.
  • Shows newly added devices in green.
  • Shows removed devices in red.
  • Shows the error number in a yellow circle on devices that has an error.

Tried with: USBTreeView 3.8.4

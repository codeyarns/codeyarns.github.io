The USB Device Tree Viewer,.also called USBTreeView, is a Windows tool that can be used to view the tree of USB controllers, hubs, ports and devices connected to those ports. Detailed information of each node in the tree is shown in the right pane.

This tool has all the features of the USBView tool, since it is based on that source code, with extra features added on top.

Some of the nice features in this tool that are not in USBView:

  • Shows the high-speed (H), full-speed (F) and super-speed (S) designation on the tree icons itself.
  • Shows the actual name of the device (and not just the device type) in the tree view itself.
  • Highlights newly added devices in green.
  • Highlights newly removed devices in red.
  • Shows the drive letter for USB storage devices.
  • Show yellow circle with exclamation (!) or error number when the device has an error. Look in the right pane for the error name associated with the error number.

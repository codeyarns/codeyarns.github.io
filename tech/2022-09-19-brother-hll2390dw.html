Brother HL-L2390DW Wireless Printer-Scanner

Brother HL-L2390DW is a B&W duplex printer and scanner all-in-one that I use at home. It be operated over its USB printer cable or over wifi if connected to the home wifi network.

  • USB printer cable: These printer USB cables are not easy to find or they are expensive. And configuring this device without hooking up over USB is a pain.

  • Brother iPrint&Scan: This app installed by the driver can be used for printing and scanning. Since I can print directly from applications, I have only used this app for scanning. I have been able to scan all sorts of B&W and color documents and save into JPEG/PDF using this app.

  • Web interface: Printer status and some basic configuration can be done using its web interface. Just type its IP address in the browser to access this.

  • Refurbished: Brother sells refurbished products on their website for good discounts. These products have a R prefix on their model name, for example RHL-L2390DW.

  • 2.5G/5G auto Wifi network: I faced problems when I switched to using a home wireless router that had auto 2.5G/5G mode. The printer could not be found intermitently by my computer for printing. Eventually, I forced the entire network explicitly to 2.5G to get around this problem.

Windows install

I downloaded the installation file for Windows from the support section of the Brother website. This was a 482MB file named Y17C_C1_UWL_PP-usa-inst-F2.exe. Executing the file runs a GUI wizard that helps wireless installation of the printer, installation of the Windows driver and the iPrint&Scan program for scanning.

Linux install

My printer had already been connected to the home network wirelessly back when I did its installation using Windows. Below are the steps I followed on my Linux computer to be able to print wirelessly to this printer:

  • Download the driver installation file from the support section of the Brother website. I chose the option Driver Install Tool which said that it would install both the printer and scanner drivers. The 24KB file I downloaded was named linux-brprinter-installer-2.2.3-1.gz.

  • Unzip the file:

$ gunzip linux-brprinter-installer-2.2.3-1.gz

This resulted in a 143KB Bash script named linux-brprinter-installer-2.2.3-1.

  • Power up the printer and make sure it is connected to the home network.

  • Run the script with sudo:

$ sudo bash linux-brprinter-installer-2.2.3-1

  • In the steps that followed, I chose Yes to specify the DeviceURI. It probed and listed the various URI and I entered the index of the Auto option listed at the end. After this, it gave me a chance to print a test page, I did that and the print was successful. It installed some .deb files and then asked me for the printer IP address. I found this IP address from the web interface of my home router and that finished the install successfully.

  • In the Printers settings of Ubuntu, I found 2 printers listed there: Brother_HL_L2390DW (Brother HL-L2390DW, driverless) and HLL2390DW (Brother HLL2390DW for CUPS). Both of them seem to be capable of printing.

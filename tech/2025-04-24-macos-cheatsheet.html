Keyboard Shortcuts

MacOS keyboard shortcuts use the special 4 keys on the Apple keyboard (L to R): fn, control, option and command.

Most of the keyboard shortcuts are visible beside the toolbar menu option when you choose them.

  • command space: Opens Spotlight, that can be used to launch applications and open files by typing their name.
  • command tab: Switch between currently open applications.
  • control command f: Maximize the current application.
  • command w: Close the window. Other windows of an application will still be open and the application will continue to run in the background (even if its final window is closed).
  • command q: Quit the application and all its windows.
  • control command q: Lock the screen

Dock

Located at the bottom of the desktop and holds the pinned applications and open applications. The running applications have a dot displayed under them. There are some default applications like Finder (leftmost icon) and Downloads and Trash (rightmost icons).

Mission Control

Spaces is the MacOS name for virtual desktops. Mission Control is the view where MacOS shows all the virtual desktops and the windows in them.

  • To open Mission Control use control up.

  • To move between spaces use control right and control left. On a MacBook, swiping 3 fingers left or right on the trackpad does the same.

  • To create a new space, open Mission Control and click the + at the top-right corner.

  • To move a window to a different space, open Mission Control and drag window to the destination space in the top bar. There are no keyboard shortcuts for this, unlike Windows.

Tiling

The options to tile windows can be found by holding the pointer over the top-left green button of the current window. In the Move & Resize section, we can choose to tile the window to the left or right vertical half. If the Displays have separate Spaces option in MacOS settings is enabled, then we get additional options here to tile the window to top or botton horizontal half or to any of the 4 quarters (need to hold down the option button to see this).

The keyboard shortcuts for moving windows to any of these 4 halves or 4 quarters can be viewed and set in Settings -> Keyboard -> Keyboard Shortcuts -> Windows.

Common operations

  • Full screen window: To toggle the current window to full screen and back use control command f.

  • Maximize window: Window can be maximized (or Zoom as MacOS calls it) by double-clicking on the titlebar. The keyboard shortcut is fn control f, but weirdly it only maximizes, but does not toggle back the window shape. I prefer going to Settings -> Keyboard -> Keyboard Shortcuts -> App Shortcuts and under All Applications, add a keyboard shortcut with Menu title Zoom and a custom keyboard shortcut such as shift command m. This keyboard shortcut both maximizes and toggles back the window shape.

  • Insert emoji: On the Macbook Pro keyboard, the Fn key brings up the emoji picker. When using a USB/Windows keyboard, pressing control command space does the same.

  • Show scroll bars: To always show a scroll bar in windows, go to System Settings -> Appearance. For Show scroll bars choose the option Always.

  • Moving in text: Macbook Pro keyboard does not have Home and End buttons. So, use command left and command right to move to start and end of current line of text. Similarly, use shift command left and shift command right to mark up to left or right end of line of text.

  • Change key repeat speed: I found it slow to move through text in editor with left-right

  • Take screenshot: Press shift command 4 to draw section of screen to capture. Thumbnail of captured image appears at bottom-right of display for editing. The image is saved to the Desktop.

  • Find IP address: Open Terminal and type ifconfig. Look for the IP address listed for the en0 network interface.

  • Lock the screen when lid is closed: Go to System Settings -> Lock Screen. Set the option Require password after screen saver begins or display is turned off to Immediately.

  • Remap Windows keyboard keys: When I connect a USB Windows keyboard, its Win key is mapped to Command and its Alt key is mapped to Option. To swap the Win and Alt key mappings, go to System Settings -> Keyboard -> Keyboard Shortcuts. In the dialog, choose Modifier Keys and in the Select keyboard option choose USB keyboard. For Option key, pick Command. For Command key, pick Option.

  • Change default mail application: Apple Mail is the default mail application. To change the default to a different mail application (like Outlook) you still need to open Apple Mail and have an account in there. Go to menu bar, choose Mail -> Settings. Go to General -> Default email reader and set to a different mail app.

  • Switch between windows: The cmd tab keyboard shortcut only switches between apps, not the individual windows, while I prefer to switch between windows (just like in Windows). AltTab is a fantastic open-source tool that can be used to switch between all windows. It can be customized to run at startup from the menubar, work with any keyboard shortcut (I like to replace cmd tab with this) and it can display the windows as thumbnails, app icons or titles, all in 3 different sizes (small/medium/large).

  • Change name of Bluetooth device: Some Bluetooth devices have associated software that can be used to change their name. If they do not, then their name to MacOS at least can be changed in the OS. To do this, open the Bluetooth settings, and click the (i) to the right of the device name. In the dialog that pops up, change the device name and press Return.

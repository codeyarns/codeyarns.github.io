Surf

2015-10-21 12:18:00+00:00

Surf is an alternate browser from the Suck Less camp. It uses the WebKit rendering engine.

It can be installed from Ubuntu archives:

$ sudo apt install surf

  • Like all Suck Less apps, this browser has no UI. It is just the page rendered inside a window.

  • Surprisingly slow. I'm guessing it is the WebKit rendering with limited threading or such.

  • Unlike the other alternate browsers, all webpages should render correctly in Surf.

Tried with: Surf 0.4.1 and Ubuntu 14.04

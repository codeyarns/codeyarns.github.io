GCP: An alternate to cp

(First posted on: 2015-01-22 12:24:20+00:00)

[caption id="attachment_5530" align="aligncenter" width="562"][](https://codeyarns.files.wordpress.com/2015/01/20150123_gcp.png) gcp showing progress of a file copy[/caption]

cp is the tool used in Linux to copy files and directories. GCP aims to be an alternate to this ubiquitous tool.

Installing it is easy:

$ sudo apt install gcp

One of its compelling features is that it shows a bar and stats of the progress of the copy operation. This can be useful when copying lots of files or to a slow disk.

An alternate to this program is Advanced Copy tool. However, that website seems to be down for sometime now.

Tried with: GCP 0.1.3 and Ubuntu 14.04

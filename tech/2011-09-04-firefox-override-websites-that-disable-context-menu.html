Firefox: Override Websites that Disable Context Menu

Ashwin Nanjappa

2011-09-04 11:12:51+00:00

Problem

Some websites use Javascript to disable or modify the usage of context menu on their webpages. Doing a right-click on these webpages to open a context menu either does not display a context menu or displays a custom context menu of that website. In all cases, such behaviour is irritating and takes control away from the user.

Solution

Overriding such disabling of the context menu for all webpages is easy in Firefox. To wrest control back from these deviant websites, head over to Tools → Options. In the Options dialog, go to Content → Advanced (Enable Javascript). In this dialog, disallow the ability of Javascript to Disable or replace context menus. That is it, it will work on all webpages! :-)

Tried with: Firefox 6

Thorium Reader

📅 2025-Feb-21 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ ebook reader, epub ⬩ 📚 Archive

I recently had to open a complex EPUB file with fixed layout of images and overlaid text. None of the regular ebook readers on Linux, like Calibre or FBReader could do the job. Thorium Reader turned out to be the one which accomplished the task perfectly. This software is open-sourced on Github here.

  • Install: I downloaded the .deb file for the latest release from here and installed it at the shell:
$ sudo dpkg -i EDRLab.ThoriumReader_3.1.0_amd64.deb
  • Sandbox error: Running Thorium for the first time, gave me the all-familiar sandbox error. Fixing that was simple:
$ sudo chown root:root /opt/Thorium/chrome-sandbox

$ sudo chmod 4755 /opt/Thorium/chrome-sandbox

Tried with: Thorium Reader 3.1.0 and Ubuntu 24.04

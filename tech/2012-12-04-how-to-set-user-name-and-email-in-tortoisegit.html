How to set user name and email in TortoiseGit

2012-12-04 07:14:36+00:00

[caption id="attachment_2954" align="aligncenter" width="781"][](http://codeyarns.wordpress.com/2012/12/04/how-to-set-user-name-and-email-in-tortoisegit/20121204-tortoisegit/#main) User name and email in TortoiseGit[/caption]

Setting the user name and email in TortoiseGit is easy.

  • Right-click anywhere in Explorer and choose TortoiseGit → Settings. In the Settings dialog, choose Git. Type in your username and email and click Apply.

  • These settings are written to the global gitconfig file for this user.

  • If you prefer using the commandline for this operation, see this post.

Tried with: TortoiseGit 1.7.15.0 and Windows 7 Enterprise x64

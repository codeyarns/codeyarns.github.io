How to set user name and email in TortoiseGit

(First posted on: 2012-12-04 07:14:36+00:00)

Setting the user name and email in TortoiseGit is easy.

  • Right-click anywhere in Explorer and choose TortoiseGit â†’ Settings. In the Settings dialog, choose Git. Type in your username and email and click Apply.
  • These settings are written to the global gitconfig file for this user.
  • If you prefer using the commandline for this operation, see this post.

Tried with: TortoiseGit 1.7.15.0 and Windows 7 Enterprise x64

I like to have Fish as my default shell in Byobu. I add these lines to my ~/.byobu/.tmux.conf:

set -g default-shell   /usr/bin/fish
set -g default-command /usr/bin/fish

Note that Byobu will set the SHELL environment variable to the above path. That is, even if you want to set SHELL to something else in your shellâ€™s init file, that will be overridden by Byobu.

Tried with: Byobu 5.74 and Ubuntu 14.04

Google Docs is the online collaborative editor from Google that tried to be a clone of Microsoft Word. Here are some tricks I find useful to know when using it:

Set default style for current doc

The default font, font size, line spacing or other formatting can be set for your current doc. This is useful so that when you add new content, it automatically follows this default style.

For example, to set the default font for normal text:

  • Highlight some normal text that is already using the font you wish to set.
  • Choose Format -> Paragraph Styles -> Normal Text -> Update Normal Text to match.

In this way you can set the style of any type of doc element. Once a style is set, all subsequent text you create in the doc will follow your style.

Set default style for all docs

The default font, font size, line spacing or other formatting can be set for all newly created docs. This is useful so that when you create a new doc, it automatically follows this default style.

To do this:

  • First set up the style in the current doc as described above.
  • Choose Format -> Paragraph Styles -> Options -> Save as my default styles.

All new docs you create now will follow this style.

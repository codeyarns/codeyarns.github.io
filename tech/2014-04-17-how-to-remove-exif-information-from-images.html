How to remove EXIF information from images

📅 2014-Apr-17 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ exif, exiftool ⬩ 📚 Archive

Using ImageMagick

The convert tool in ImageMagick can be used to strip out all EXIF information from an image like this:

$ convert in.jpg -strip out.jpg

Using EXIFTool

The EXIFTool tool can be installed from the libimage-exiftool-perl package:

$ sudo apt-get install libimage-exiftool-perl

To remove all EXIF tags from an image file:

$ exiftool -all= foo.jpg

The foo.jpg has been stripped of all EXIF tags. The original file is saved for backup as foo.jpg_original.

Tried with: ExifTool 8.60-2 and Ubuntu 12.04

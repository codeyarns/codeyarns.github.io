View Ebooks with Calibre

2012-02-26 01:11:42+00:00

Calibre is a huge tool that can be used to manage, view and convert all kinds of ebook formats. But, if you just want a simple tool to view ebooks (like EPUB, MOBI, AZW3 and other formats), use the E-Book Viewer (ebook-viewer.exe) tool that ships with Calibre.

Tried with: Calibre 0.8.41

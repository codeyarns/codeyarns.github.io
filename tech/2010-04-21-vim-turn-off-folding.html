ViM: Turn Off Folding

📅 2010-Apr-21 ⬩ 📚 Archive

ViM automatically turns on folding for certain files. I do not like folding and keep it disabled all the time by adding this to my vimrc:

set nofoldenable

To disable only for the current session type:

:set nofoldenable

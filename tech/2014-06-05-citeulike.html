CiteULike

(First posted on: 2014-06-05 03:43:12+00:00)

CiteULike is a website to maintain references to papers.

Advantages

  • Provides a browser bookmarklet that makes it easy to add a reference from the webpage of any paper.

  • Tags

  • Easy to generate BibTeX entry from any saved reference.

  • Can store the PDF of paper along with the reference, so you can access it from anywhere.

Disadvantages

  • Cannot edit the BibTeX entry it maintains for a paper.

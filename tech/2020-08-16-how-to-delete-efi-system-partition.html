How to delete EFI System Partition

📅 2020-Aug-16 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ diskpart ⬩ 📚 Archive

I was given a USB flash drive that had a FAT drive which appeared to be a EFI System Partition when viewed in the Windows Disk Management tool. I wanted to delete this partition, but the disk management tool did not provide any option to resize or delete this partition.

To delete the EFI System Partition, we need to wipe the partition table of this flash drive. To do that:

  • Insert the flash drive to computer and copy over all the files.
  • Open a command prompt in Administrator mode.
  • Type diskpart to open the commandline disk partition tool.
  • List the disks connected to your computer using the command: list disk. Carefully note down the disk number of the flash drive. In my case, it was number 2.
  • Select the disk using the command: select disk 2. Replace 2 with your disk number.
  • Type clean to wipe the partition table of the disk.
  • Type exit to exit the disk partition tool.
  • After this I was able to create any partition I wanted on the flash drive using the GUI Disk Management tool.

Tried with: Windows 10

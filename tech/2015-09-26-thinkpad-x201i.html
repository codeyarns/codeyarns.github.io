Thinkpad X201i

📅 2015-Sep-26 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ thinkpad ⬩ 📚 Archive

I’m using a Thinkpad X201i notebook for the next few months. This post is a summary of the notes for this notebook. This will be updated as and when I learn more about this notebook.

  • Fn key on the keyboard of this notebook is placed to the left of the Ctrl key. This is quite irritating if you are used to having the Ctrl key as the bottom-left key on the keyboard. Thankfully, these keys can be swapped in the BIOS setup of this notebook. The change will persist in any OS you use without requiring any software configuration.

  • Fn + PgUp: To toggle the keyboard light. This is very useful when working in the dark.

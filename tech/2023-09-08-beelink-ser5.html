The Beelink SER5 is a popular series of mini PC featuring AMD CPUs. I got the mini PC with the AMD Ryzen 5500U recently.

Usage

  • F7: Get boot menu
  • Del: Get into BIOS

Installing hard disk

The Beelink came with a manual showing how to install an extra hard disk inside the case. However, most of those instructions related to attaching the cable were unnecessary since the cable was already attached.

Here is what I had to do:

  • Power off the Beelink, unplug everything and flip it over.
  • Remove the 4 screws at the bottom and using the fabric tab pull up the cover gently.
  • Slide your hard disk or SSD into the slot and press until it locks into the interface inside.
  • You can secure it with up to 4 screws provided in the box, but the holes were slightly misaligned with my hard disk so I could not do it.
  • Close the cover and screw in. Reboot and enjoy.

Dual booting with Linux

This was pretty straightforward except for some problems I faced since I had added an extra hard disk that used to be a bootable Linux OS earlier.

Here is what worked:

  • Create a Linux installer USB flash drive as described here.
  • Restart the mini PC, choose F7 to get the boot menu and choose to boot from the flash drive.
  • Install Ubuntu. Either using the along with Windows boot manager option or you can create an ext4 / partition and use that. Remove the flash drive and reboot.
  • If you do not see the Linux/Grub boot menu, but instead directly boot into Windows 11, then check the steps here.

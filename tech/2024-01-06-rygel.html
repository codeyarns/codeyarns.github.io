Rygel is a UPNP/DLNA media server for Linux. In Ubuntu or GNOME, it is the default media server and is available under Media Sharing.

  • Install: Rygel should already be installed on Ubuntu. In case it is not, it can be installed easily:
$ sudo apt install rygel

  • Enable: Go to Settings -> Sharing -> Media Sharing. Enable it, specify which video, music and photos directories to share and choose the home network to share on. That is it!

  • Access: Open any media player on any device on your home network that supports UPNP/DLNA and you should be able to access your media. For example, VLC on iPhone shows the Rygel server under the Network -> File Servers section of the app.

  • Configuration: Edit the /etc/rygel.conf file to change the settings.

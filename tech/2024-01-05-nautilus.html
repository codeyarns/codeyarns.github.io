Nautilus is the official file manager in GNOME and Ubuntu. It is also called the Files application in these desktop environments.

  • Bookmarks: Directories that are bookmarked appear listed at the bottom of the sidebar. To add a directory as bookmark, select it and drag it to the bookmarks list and drop it over New bookmark that appears there. Or press Ctrl + d when inside a directory to add it to the bookmarks. The bookmarks can be reordered by dragging them up or down.

  • Tabs: Nautilus does not support horizontal or vertical tiled panels. Though not an ideal solution, tabs can be used instead. Press Ctrl + t to open a new tab. Files can be dragged between tabs.

  • Zoom the files: Hold down Ctrl and scroll the mouse wheel to change the size of the files displayed. This works in both grid and list views.

  • Jump to directory: The path displayed in the addressbar is uneditable using any mouse operation. Instead press Ctrl + l and it becomes editable to enter any path you want.

    • Press / as shortcut to jumping to the root directory.
    • Press ~ as shortcut to jumping to the home directory.

  • Toggle hidden files: Nautilus considers files and directories that begin with . or end with ~ as hidden. To toggle viewing or not viewing hidden files press Ctrl + h.

References

  • https://help.gnome.org/users/gnome-help/stable/files.html.en

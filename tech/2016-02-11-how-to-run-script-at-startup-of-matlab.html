How to run script at startup of MATLAB

Ashwin Nanjappa

2016-02-11 12:12:39+00:00

It is sometimes useful to run a MATLAB script when it is started. For example, to setup the environment just the way you want before you begin your work.

MATLAB supports this usage. When you start MATLAB from the shell, it looks for a startup.m script in the current directory and executes the commands in it after starting up. The directory from which MATLAB is started is called the MATLAB Startup Folder.

So, create a startup.m with the commands you need and run MATLAB from that directory.

Reference: startup

Tried with: MATLAB R2014a and Ubuntu 14.04

If your Windows computer supports Bluetooth, then it must have a Bluetooth chip that provides this capability, typically called as a Bluetooth radio. The Bluetooth version or more specifically the version of the Bluetooth Core Specification supported by this radio determines what features of Bluetooth are available on your computer and for the BT devices you connect to it. But surprise, Windows does not display this Bluetooth version anywhere!

To find out the Bluetooth version:

  • Find your Bluetooth radio: Open Device Manager by typing device manager and go to Bluetooth -> â€œName of your Bluetooth radioâ€. On my computer, the Bluetooth support is provided by Intel and the radio was named Intel Wireless Bluetooth.
  • Find its LMP version: Right-click on it, choose Properties and open the Advanced tab. Note down the LMP (Link Manager Protocol) version listed under Firmware version information. Mine said HCI 10.256 / LMP 10.256.
  • Convert LMP to Bluetooth version: Convert the LMP version to the Bluetooth Core Specification version using the table provided here or here. Mine turned out to be Bluetooth version 5.1.

BVHPlay

2013-08-24 11:26:18+00:00

BVH (BioVision Hierarchy) is a file format used for motion capture files. BVHPlay is a bare-bones simple program that can be used to play and view BVH files. It can be downloaded here.

BVHPlay is written in Python and the source code is available. The code is well written and is useful understand how to play back the frames of a BVH file yourself.

Tried with: BVHPlay 1.00 and Ubuntu 12.04 LTS

Save Page WE Firefox Extension

📅 2023-Dec-22 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ firefox extension ⬩ 📚 Archive

Save Page WE is an extension for Firefox that enables saving a webpage into a single file that can be opened in any browser. Built into Firefox is the ability to save “Web Page, HTML only” (which saves only the HTML portion into a single file) and “Web Page, complete” (which saves into a HTML file, plus a directory of the webpage files). This extension enables saving all the page content like the complete feature but inside a single file like the HTML only feature.

  • Install: This can be installed from here.

  • Usage: Right-click on the current webpage and choose Save Page WE -> Save Selected Tabs and choose one of 3 options: Basic Items, Standard Items or Custom Items. This saves into a single HTML file.

Tried with: Save Page WE 28.11, Firefox 119.0.1 and Ubuntu 22.04

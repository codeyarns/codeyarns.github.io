Matlab commands list

(First posted on: 2016-02-08 12:58:44+00:00)

  • addpath: Command to add a directory to path

  • dbquit: Exit debug mode

  • doc: Open documentation browser

  • fieldnames: Get field names of a struct.

  • lookfor: Show one-line help. Like apropos.

  • help: Print help about input function. Like man.

  • path: Show search path. These are the directories from which MATLAB will run scripts.

  • prefdir: Show directory where MATLAB stores preferences files

  • profile viewer: GUI of profiler

  • pwd: Show PWD

  • why: Funny answers to profound questions

  • workspace: Open Workspace

  • wrldtrv: Great circle flight routes for the globe

  • xpbombs: Play minesweeper

How to use color themes in Eclipse

If you are getting bored of the default syntax highlighting, foreground and background colors in Eclipse, it might be time to liven things up a bit by trying a different color theme. The most convenient way to apply and change color themes is by using the Eclipse Color Theme Plugin. Installing and using it is easy:

  1. Go to Help > Install New Software.

  2. In the Work With box use http://eclipse-color-theme.github.com/update, click Add and choose to install the plugin.

  3. Restart Eclipse after plugin is installed.

  4. To change the theme, go to Window > Preferences > General > Appearance > Color Theme.

The plugin comes with a few themes. And there are plenty more that can downloaded at the Eclipse Color Themes website.

Tried with: Eclipse 3.7.2, NVIDIA Nsight 5.5 and Ubuntu 12.04

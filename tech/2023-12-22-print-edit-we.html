Print Edit WE is an extension for Firefox that makes it super easy to edit the contents of the webpage currently being displayed. This editing UI/UX/controls of this extension make it truly easy to quickly delete all the unnecessary parts of the webpage and focus on the content. This can be useful for distraction-free reading or for printing.

  • Install: The extension can be installed from here.

  • Usage: Click the extensionâ€™s button in the toolbar to enable it on the current webpage and display its buttons.

    • Hover the mouse on the webpage elements and they are highlighted with a red border.
    • Click on any element to select it and operate on it using the extensionâ€™s buttons.
    • Once all the non-essential elements are removed, you can now read the webpage.
    • To print, first do a print preview of this simplified page. And if you are satisfied, then print it.

  • Buttons:

    • Hide or Delete: Hide or delete the currently selected webpage element.
    • Hide except or Delete except: Keep the currently selected element and remove everything else.

Tried with: Print Edit WE 29.6, Firefox 119.0.1 and Ubuntu 22.04

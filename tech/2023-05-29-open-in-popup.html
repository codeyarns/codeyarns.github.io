I was looking for a way to preview the webpages pointed to by links on hover in Firefox. I tried many extensions and in the end had to choose Open in popup written by Soushi Atsumi.

With this extension I had to right-click on the link and choose an option, but it was the only addon that did the job. It opens the webpage behind the link in a popup window that can even be scrolled.

Default keyboard shortcut to do the same is Shift + Alt + O.

Alternatives

Listed below are the other addons that did not work:

  • Less Simple Link Preview
  • Link Peek: Did not work on most webpages.
  • Peek Preview: Only does preview of documents like PDF, not of webpages.
  • Survol

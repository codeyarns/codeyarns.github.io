LaTeX Error: Font expansion error

2014-05-19 08:58:26+00:00

Problem

You compile a LaTeX document and get this error:

pdfTeX error (font expansion): auto expansion is only possible with scalable fonts.

Solution

Install the cm-super fonts package to fix this error:

$ sudo apt-get install cm-super

Tried with: Ubuntu 14.04

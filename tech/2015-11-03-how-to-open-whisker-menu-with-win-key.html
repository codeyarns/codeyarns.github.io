How to open Whisker Menu with Win key

(First posted on: 2015-11-03 14:13:10+00:00)

Whisker Menu in XFCE is the equivalent of the Start menu in Windows. You get it by clicking the mouse icon in the top-left corner.

The default keyboard shortcut to activate this is Ctrl + Esc. For anyone who has used Windows or Unity before, this can be an irritating shortcut to remember. Why not just set this to the Win key?

That is easy. Go to Settings -> Keyboard -> Application Shortcuts. For the command xfce4-popup-whiskermenu, change it to the Win key.

Tried with: XFCE 4.12 and Ubuntu 15.04

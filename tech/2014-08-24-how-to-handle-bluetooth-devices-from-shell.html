How to handle Bluetooth devices from shell

Bluetooth devices can be listed, connected and disconnected by clicking the Bluetooth icon in the top Panel in the Ubuntu Unity desktop. If you prefer to do these simple operations from the commandline, that can be done too.

Ubuntu 22.04

  • Install the bluez package that has many Bluetooth tools:
$ sudo apt install bluez
  • The tool we need to use is bluetoothctl and to get help about its usage:
$ bluetoothctl help
  • To list the Bluetooth controllers on your computer:
$ bluetoothctl list

This lists the UUID and user-friendly name of each controller.

  • To list detailed info about the Bluetooth controllers:
$ bluetoothctl show
  • To list the Bluetooth devices paired to this computer:
$ bluetoothctl paired-devices

This lists the UUID and user-friendly name of each Bluetooth device.

  • To list detailed info about a Bluetooth device:
$ bluetoothctl info <UUID of device>

If no UUID is provided, it prints the info for only the connected Bluetooth devices.

  • To connect a Bluetooth device:
$ bluetoothctl connect <UUID of device>

Note that in the Fish shell, I can type a few letters of the device name and have Fish tab-complete it to the matching UUID.

  • To disconnect a Bluetooth device:
$ bluetoothctl disconnect <UUID of device>

Note that in the Fish shell, I can type a few letters of the device name and have Fish tab-complete it to the matching UUID.

Ubuntu 14.04

  • To do this, we need the tools from the bluez-tools package:
$ sudo apt install bluez-tools
  • To list all the devices that have paired in the past:
$ bt-device --list

This lists each device with its name and UUID. The UUID of a device can be used to connect or disconnect from it.

  • To connect an audio device using its UUID, say 00:02:3C:2F:F1:D4:
$ bt-audio --connect 00:02:3C:2F:F1:D4
  • To disconnect the same audio device:
$ bt-audio --disconnect 00:02:3C:2F:F1:D4

Tried with: Bluez-tools 0.1.38 and Ubuntu 14.04

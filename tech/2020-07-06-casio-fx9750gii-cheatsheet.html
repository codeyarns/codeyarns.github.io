Casio fx-9750GII Cheatsheet

📅 2020-Jul-06 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ calculator, casio, cheatsheet ⬩ 📚 Archive

Math mode

  • EXE: To perform the calculation and display the result. Equivalent to = on older calculators.
  • AC: To clear the display of previous calculations.
  • To use previous answer as first operand in current calculation: just skip typing in the operand and type the rest of the expression. Example: to be able to calculate Ans x 3 =, type in x 3 EXE.
  • ↑ or ↓: To cyle through previous and next calculations. At a particular calculation, edit it using → or ←.
  • Shift EXP: To insert π.

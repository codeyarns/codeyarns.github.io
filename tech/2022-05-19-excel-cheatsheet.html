Microsoft Excel is the popular spreadsheet program from the Microsoft Office suite.

  • Excel is available as a Windows desktop application called Excel for Windows. It is also available in the browser as Excel for the web.

  • It is one of the many tools that are collectively available under the subscription based program called Microsoft 365.

  • The Excel document is called a workbook and the sheets inside are called worksheets.

  • Most of the advanced options are not available in Excel for the web. When an advanced option is required, choose Editing (top-right corner) -> Open in Desktop App. This opens the workbook in Word for Windows and can be edited with access to all the tools.

  • Change default font: Changing the default font is not possible in Excel for the web. In Excel for Windows, open Options. In the dialog, go to General -> When creating new workbooks and change the font for the option Use this as the default font.

  • To switch to next/previous sheet: Use Ctrl PgUp or Ctrl PgDown to switch to next/previous sheet.

  • To move a row or column: Select the entire row or column by clicking on row name on the left or the column name at the top. While holding down Shift on the keyboard, move mouse cursor to the top/bottom (of row) or left/right (of column) border of selected row or column cells (not the left or top bar). If you did this correctly, the mouse cursor will turn into ‚ú•. Still holding down Shift, left-click the mouse and drag the row or column to where you want it to be and let go. Note: This only worked with the Windows app and not in the browser version of Excel.

  • Move row(s) using keyboard: Press Shift Space to select the entire row you want to move. (If you want to move multiple rows, hold down Shift and press Up or Down to select all the rows you want to move.) Press Ctrl x to cut this selected row. Use keyboard to go to the row above which you want to insert the row we cut. Press Ctrl Shift + to insert the row.

  • To paste as plain text: Press Ctrl Alt v

  • To insert row above current row: Press Ctrl Shift +.

  • To scroll horizontally: Hold down Ctrl Shift while using the mouse scroll wheel. For some reason, this scrolls a lot more than the visible columns for me. If your mouse has a special horizontal scroll wheel, that is better to use. (My Logi MX Master 3 horizontal scroll wheel scrolls with high precision column-by-column.)

  • Insert date or time: To insert the current date or time as a static value, press Ctrl ; (for date) and Ctrl Shift ; (for time).

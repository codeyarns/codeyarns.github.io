How to find version of Raspbmc

📅 Published on: 2012-Nov-09

Nowhere in its XBMC UI does Raspbmc reveal its version number. If you need to find out the version number, SSH into it and check this obscure file:

$ cat /scripts/upd_hist/build_info

Thanks to Steve Robillard for this information.

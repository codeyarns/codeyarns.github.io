How to reverse page order of PDF using PDFTK

Ashwin Nanjappa

2012-05-13 00:17:12+00:00

PDFTK can be used to reverse the order of pages in a PDF file.

To install PDFTK, please follow the instructions here.

To reverse the 20 pages of a PDF file named in.pdf and write to out.pdf:

$ pdftk in.pdf cat 20-1 output out.pdf

Tried with: PDFTK 1.41

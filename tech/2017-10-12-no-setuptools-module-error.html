Problem

Installing a package with Pip on a new computer gave this error:

ImportError: No module named 'setuptools'

Solution

For Python2:

$ sudo apt install python-setuptools

For Python3:

$ sudo apt install python3-setuptools

Tried with: Ubuntu 16.04

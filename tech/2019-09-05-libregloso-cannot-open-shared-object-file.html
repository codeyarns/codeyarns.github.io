libreglo.so cannot open shared object file

2019-09-05T22:22:27.339445-07:00

Problem

I invoked LibreOffice from the shell and got this error:

$ soffice
/usr/lib/libreoffice/program/javaldx: error while loading shared libraries: libreglo.so: cannot open shared object file: No such file or directory
Warning: failed to read path from javaldx
/usr/lib/libreoffice/program/soffice.bin: error while loading shared libraries: libreglo.so: cannot open shared object file: No such file or directory

Solution

Add the path containing LibreOffice library files to your LD_LIBRARY_PATH and it will solve this problem:

$ export LD_LIBRARY_PATH="/usr/lib/libreoffice/program:$LD_LIBRARY_PATH"

How to open WebP file in GIMP

2014-04-11 03:06:16+00:00

WebP is an image format developed by Google. Most image viewers and editors do not support it right now. To be able to open WebP image files in GIMP, install the gimp-webp and libwebp packages from the george-edison55/webp PPA:

$ sudo apt-add-repository ppa:george-edison55/webp
$ sudo apt-get update
$ sudo apt-get install gimp-webp libwebp

Tried with: GIMP 2.6.12, LibWebP 0.1.3.3 and Ubuntu 12.04

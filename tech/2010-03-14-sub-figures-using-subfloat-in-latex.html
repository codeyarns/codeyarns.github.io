Sub-figures using subfloat in LaTeX

(First posted on: 2010-03-14 00:33:04+00:00)

[caption id="attachment_2689" align="aligncenter" width="628"][
](http://codeyarns.files.wordpress.com/2010/03/20120731-subfloat.png) Two subfigures in a figure in LaTeX[/caption]

To include more than one figures inside a figure the subfig package can be used. It provides the subfloat command that is used to specify each of the sub-figures as a figure. Its usage is shown below in a simple example:

https://gist.github.com/3214225

Note that each of the subfigures can be given an optional label which can be used to refer to them in the text of the document.

If you are using TeX Live, in Ubuntu for example, subfig can be found in the texlive-latex-recommended package. Note that I have seen that subfig has problems working with Beamer.

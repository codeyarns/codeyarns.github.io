How to hide dot files in Filezilla

📅 2014-May-12 ⬩ ✍️ Ashwin Nanjappa ⬩ 📚 Archive

[caption id="attachment_4570" align="aligncenter" width="640"][](http://codeyarns.files.wordpress.com/2014/05/20140512-filezilla-dotfiles.png) Hide dot files and directories in Filezilla[/caption]

By default, Filezilla displays all the dot files and dot directories. These files are typically hidden by most file explorers. If you find the display of these files and directories irritating, you can hide the dot files and dot directories using a filter.

  • Go to View > Filename filters. This displays a dialog of Directory listing filters.

  • Click on Edit filter rules. This displays a dialog of Edit filters.

  • We add a new filter here to hide dot files. Click on New and give the filter a name. In Filter conditions dropdown choose Filter out items matching all of the following. Choose Filename, begins with and add a period (dot). Make sure the checkboxes at the bottom for Filter applies to are set to both Files and Directories. Click OK.

  • You are back in the Directory listing filters dialog. Enable the filter you just created and the dot files should be hidden! 😊

Tried with: Filezilla 3.7.3 and Ubuntu 14.04

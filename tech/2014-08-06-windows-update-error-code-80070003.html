Windows Update error code 80070003

(First posted on: 2014-08-06 14:33:02+00:00)

Problem

I tried to install the updates in Windows Update and it reported that the update failed and the error code is 80070003.

Solution

This error is caused by the update stuck at some temporary files. The steps to fix this is provided here. It is simple:

  • Stop the Windows Update service.

  • Delete all the files and directories inside C:\Windows\SoftwareDistribution\DataStore

  • Start Windows Update service back.

Tried with: Windows 7 x64

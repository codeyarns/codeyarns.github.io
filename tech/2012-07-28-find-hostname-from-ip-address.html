Find hostname from IP address

(First posted on: 2012-07-28 05:30:05+00:00)

Problem

You know the IP address of a computer. You want to find out the name of the computer or its hostname.

Solution

The humble ping command can resolve the hostname from a IP address. Use the ping -a 172.20.30.40 command to find the hostname of the computer with IP addressÂ 172.20.30.40.

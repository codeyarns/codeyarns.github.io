MP3Tag is a popular and useful MP3 tag editor on Windows. I was looking for an alternative to MP3Tag on Ubuntu and discovered PuddleTag.

PuddleTag tries to be a drop-in replacement for MP3Tag and it gets the job done. It can be used to edit tags and filenames of MP3 and FLAC files.

  • Install: Installing PuddleTag is easy:
$ sudo apt install puddletag

  • Usage: Open the MP3 files so that they are listed in the app. Manipulate their tags and save to ID3 tags or filenames.

  • Cover art: By default, the cover art window is not visible. Choose Windows -> Artwork to display this window on the right side. Right-click on the black square or cover art to find the options to add or change it.

Tried with: PuddleTag 2.0.1 and Ubuntu 22.04

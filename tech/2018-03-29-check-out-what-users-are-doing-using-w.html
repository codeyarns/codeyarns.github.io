Check out what users are doing using w

(First posted on: 2018-03-29 01:38:48+00:00)

w is a good command to get an instant status of a Unix computer. You can get the uptime, system load, list of users, when they logged on and what they are doing.

  • To get this tool, you might need to install the procps-ng package. I found that this package name is the same on both Ubuntu and Cygwin.

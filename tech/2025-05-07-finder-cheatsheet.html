Finder Cheatsheet

📅 2025-May-07 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ finder, macos ⬩ 📚 Archive

The Finder is the file manager in MacOS. Its logo is a smiling face and called the Happy Mac.

  • Cut and paste files: Files cannot be cut in Finder. Instead, copy the files (command c) and while pasting in the destination directory use option command v. This special paste moves the files instead of copying them over.

  • Quit Finder from Dock: Finder always seems to be running in the Dock. Apparently, it cannot be quit and will always be running in the Dock. Even when killed, it will be respawned.

Tried with: Finder 15.3 and MacOS Sequoia 15.3

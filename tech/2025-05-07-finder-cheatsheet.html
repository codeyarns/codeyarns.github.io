Finder Cheatsheet

📅 2025-May-07 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ finder, macos ⬩ 📚 Archive

The Finder is the file manager in MacOS. Its logo is a smiling face and called the Happy Mac.

  • Search: To search for files or directories, click the 🔍 or press command f. Finder searches the entire Mac by default. To restrict the search under the current folder, click the current folder name in the Search field.

  • Cut and paste files: Files cannot be cut in Finder. Instead, copy the files (command c) and while pasting in the destination directory use option command v. This special paste moves the files instead of copying them over.

  • Quit Finder from Dock: Finder always seems to be running in the Dock. Apparently, it cannot be quit and will always be running in the Dock. Even when killed, it will be respawned.

  • Hide disks from desktop: Go to Finder -> Settings -> General. Under Show these items on the desktop, uncheck all the types of disks and servers you would like to be hidden on the desktop.

  • View path bar: To display the path of any directory that Finder is showing, choose View -> Show Path Bar. Finder shows the path at the bottom, and makes it easy to click and jump to any directory that is part of that path. Note how this is different from Windows Explorer, which shows the path at the top.

  • Mount Samba share: Samba is a convenient way to share a directory from a local Linux computer to our Mac. To access that Samba share in Finder, choose Go -> Connect to Server and use smb://<local-ip>. You will be asked to enter the login and password of that Samba share. Once added this local computer will be listed under Locations on the left side.

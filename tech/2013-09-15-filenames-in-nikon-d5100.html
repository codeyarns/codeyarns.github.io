Filenames in Nikon D5100

(First posted on: 2013-09-15 14:34:27+00:00)

Nikon D5100 uses two types of filename prefix: DSC and _DSC. The latter is not documented in their manual and may surprise some users when they see filenames using that prefix.

DSC is used if you have set the Color Space to sRGB.

_DSC is used if you have set the Color Space to Adobe RGB.

To change the Color Space go to Menu > Shooting Menu > Color space

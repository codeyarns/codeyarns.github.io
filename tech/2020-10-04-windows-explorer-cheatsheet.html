Windows Explorer Cheatsheet

📅 2020-Oct-04 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ cheatsheet, windows explorer ⬩ 📚 Archive

Here are some keyboard shortcuts and usage tricks in Windows Explorer that I find useful:

  • Backspace or Alt + Left: Jump back to previously open path. Use this repeatedly to walk back through previously open directories.
  • Alt + Up: Open parent directory. Use this repeatedly to move up in the directory hierarchy.
  • Ctrl + L or Alt + D: Jump to address bar and highlight the current path displayed there.
  • Ctrl + Shift + N: Create a new subdirectory (or folder) in the currently open directory.

Pomodoro on Ubuntu using Tomate

📅 2013-Mar-23 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ timer ⬩ 📚 Archive

Pomodoro is a popular technique to work in focused stretches of time with high quality results. If you use the Pomodoro technique, then Tomate is a good Pomodoro timer for you that runs on Ubuntu.

To install it, add the PPA repository ppa:stvs/tomate and install from the repository using the package name tomate:

$ sudo add-apt-repository ppa:stvs/tomate
$ sudo apt-get update
$ sudo apt-get install tomate

Tried with: Tomate 0.2 and Ubuntu 12.04.2 LTS

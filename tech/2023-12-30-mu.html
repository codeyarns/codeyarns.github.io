Mu Python IDE

📅 2023-Dec-30 ⬩ ✍️ Ashwin Nanjappa ⬩ 🏷️ python ⬩ 📚 Archive

Mu is a simple Python IDE created by Nicholas H. Tollervey that is designed for beginners and students. Most Python books and tutorials recommend IDLE, but I have found it to be outdated and ill-suited when teaching my kid. Also, IDLE does not work well on Windows or in WSL.

Mu solves all these problems, plus it installs all the common modules required by students (like PyGame and Turtle) by default. Also, it can be used like a debugger since it support placing breakpoints. In the 2 years of struggling to interest my kid in learning Python, I have found that this is the best solution so far.

  • Install: Mu works on Windows, Linux and MacOS. Download the installer for your OS from here and install it.

  • Open source: The source code for this IDE can be found at: https://github.com/mu-editor/mu

  • First run: When you run it for the first time, it creates a virtual environment and installs a lot of common modules required by students. So expect it take a short while on the first run.

  • Modes: Mu supports different modes suited for different kinds of projects. I have used the Python 3 mode with my kid. Some of the other modes include Pygame Zero and Snek. Mu automatically installs the modules required for the chosen mode.

  • Source files: On Windows, the source files created by the users are stored in the %USERPROFILE%\mu_code directory.

Tried with: Mu 1.2.0 and Windows 11 x64
